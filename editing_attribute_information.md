**Editing Attribute Information**


 


An attribute is created in the Master Attribute List with basic characteristics relating to the attribute. Some of these characteristics can be modified to reflect the specific characteristics required of that attribute for a particular category. These characteristics include:


- whether the an attribute value is required to be entered (no nulls) in this field
- whether a range of values can be set for this attribute as it applies to this category
- the unit of measure for a numeric attribute can be changed for this category

 


An approved attribute cannot be modified.


 


An approved attribute can be set as **Required**. This designates that the attribute value must be entered for each item in the category; it cannot have a null value. In the Data Developer module, the column heading for a required attribute will be displayed in red type.


 


**The category developer role is required to be able to edit attributes in a category.**


 


**To set an attribute as Required:**


 


- From the Classification Manager window, navigate through the classifiction to the category to be modiifed

 


- Select the category to display the category information in the right hand column

 


- Select the **Attributes** tab

 


- Select the attribute to be set to Required

 


- Right click on the mouse and select **Edit...**

The **Attribute Editor** window will be displayed


 


- Under the **Validation** heading, select the Required checkbox

 


- Select **Save**
