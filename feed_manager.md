# Feed Manager


The SmartFeeds **Feed Management** page is where you can see an overview of all configured feeds.


 


Navigate to the **SmartFeeds module** to view.


![](assets/image(136).png)




 


From this view, you will be able to see:


![](assets/image(137).png)




**A. Feed Name**


**B. Feed Description**


**C. Last Ran Date -** The date the import feed was last run


**D. Type -** The type of feed (import feed vs deliverable import feed)


**E. Status -** The status of the most recent feed run (Successful or Failed)


**F. Fail Severity -** The level of fail severity set for the feed


**G. Trigger -** The condition that makes the feed run


**H. Is Active -** A check mark means the feed is currently active


**I. Options -** Delete feed, Edit feed, or Manage Mappings


**J. External Model -** The external data model being used by the feed


 


At the top right of the page, you can create new feeds by clicking the relevant button.
