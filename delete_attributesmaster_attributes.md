# Delete Attributes(Master Attributes)


SmartClass gives the user the ability to delete an attribute across all categories. Deleting an attribute should only be done when there are duplicates or an unnecessary attribute in the attribute list. The user must be careful when deleting attributes because once deleted, the attributes cannot be salvaged. 


 


In the Attributes page, locate the desired attribute to be deleted; the user may search by page or by using the filter. 


Select the attribute.


![](assets/image(707).png)




 


Click on Operations and select Delete.


![](assets/image(708).png)




 


A final message will appear confirming the deletion. 


Select Delete.


![](assets/image(499).png)




 


The attribute has been deleted from the master list. This cannot be undone.
