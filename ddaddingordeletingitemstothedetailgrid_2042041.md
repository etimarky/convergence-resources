**Adding 
or Deleting Items to the Detail Grid**



To 
add items to the detail grid, right click in the lower grid, and choose “Add 
Item”.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Adding-or-Deleting-Items-to-the-Detail-Grid_files/image001.jpg)




**Example**: Adding an item to 
the detail grid.


 


After 
selecting Add Item, the window displays enabling the user to search for any 
item(s).  The dialog box is displayed for the user to select the item they wish 
to add.  Locate the appropriate item(s), and choose Ok.


 


 


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Adding-or-Deleting-Items-to-the-Detail-Grid_files/image002.jpg)




**Example**: Adding 3 items to 
the detail grid.


 


*Note*: To add items to the 
detail grid, the user must be the assigned user for the main item selected in 
the main grid.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Adding-or-Deleting-Items-to-the-Detail-Grid_files/image003.jpg)




**Example**: Deleting an item 
from the detail grid.


 


To 
delete items from the detail grid, select the item, right click, and choose 
“Delete Item”


 


Much 
like the upper grid, the detail grid also allows for assigning to new batches, 
setting status, and assigning items.


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Adding-or-Deleting-Items-to-the-Detail-Grid_files/image004.jpg)




**Example**: Various commands 
for the detail grid
