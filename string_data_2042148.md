**String Data**


 


String attributes apply to alphanumeric values where the values could include alphabetic characters. An attribute called "color" may have alpha values of blue, green and white.  A string attribute could also be a combination of alpha and numeric characters. 


 


An attribute called "supplier part number" may have values: 123RDH, 45-TU-67.  


 


String types do not have units of measure associated with them.
