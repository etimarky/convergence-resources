



User\_Manager - Design For Retrieval (DFR) Help




**Remove 
Batch**


 


Remove 
Batch is a tool off the main menu Tools>Remove Batch to Remove Loader and 
Working batches.  
  



![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/RemoveBatch(LoadOffRoot)_files/image001.jpg)




This 
brings the user to the Remove Batch dialog box.  This removes Loader batches 
and Working batches.  This depends on the “sandbox” setting in Options Manager.  
If sandbox mode is set to ‘on’ or true, the user can delete working batches and 
load batches.  Both batch types WILL show up in the list below.  If sandbox 
mode is set to ‘false’ or ‘off’, only working batches will show up in the list 
below.  Roles have no affect on deleting privileges or whether Load batches or 
Working batches show up in the list.  This is driven by the Sandbox mode 
setting in Options Manager.  Options Manager requires the role of Catalog 
Administrator to change any setting in Options Manager.   


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/RemoveBatch(LoadOffRoot)_files/image002.jpg)




Click 
on the ![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/RemoveBatch(LoadOffRoot)_files/image003.jpg)

 and 
this will delete the working batch with no more confirmations asking the user 
if they want to delete it.  The system will delete the working batch.  


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/RemoveBatch(LoadOffRoot)_files/image004.jpg)




The 
Working batch is deleted only meaning the ‘grouping’ of items is deleted.  


If 
sandbox mode is set to true, the user will see load batches.  


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/RemoveBatch(LoadOffRoot)_files/image005.jpg)




Click 
on the ![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/RemoveBatch(LoadOffRoot)_files/image006.jpg)

 and 
this will delete the loader batch which includes all the parts.  Since the 
system is deleting parts, it gives the user a message asking for confirmation.  



![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/RemoveBatch(LoadOffRoot)_files/image007.jpg)




It 
will confirm how many items were deleted in the loader batch.
