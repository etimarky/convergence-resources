**Exporting**


 


The 
Data Developer allows users to export data to an excel or CSV format.  Any data retrieved in Data Developer, 
independent of assignment, can be exported.  
There are a few options to exporting items to consider.


  
 
 


 


**Exporting Select Items:**


 


- Retrieve the item or items you want to export 
 from Data Developer.
- Select the first column and select the number of 
 rows of items you want to export.  A 
 user can select items individually using the control key or they can 
 select a group of items using the shift key to highlight multiple rows.
- Once your rows are selected, now go to the 
 reports menu option and select one of two options for exporting: excel or 
 CSV.
- A dialogue will come up, asking if you want to 
 export select items, select yes.
- Enter a file name and save your exported file.

**Exporting All Items:**


 


·         
Retrieve the item or items you want to export from Data Developer.  
  



·         
Go to the reports menu option and select one of two options for 
exporting: excel or CSV.  
  



·         
Enter a file name and save your exported file.


 


 


For 
both exporting options, any columns that are hidden in the grid will not get 
exported.  The data will be exported in 
the same order that attributes are in Data Developer.
