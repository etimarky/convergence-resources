# Create and Delete Attribute Groups

Create\_and\_Delete\_Attribute\_Groups - Design For Retrieval (DFR) Help

## Create and Delete Attribute Groups

&#x20;

In the Attribute Groups page, click on "Add New Attribute Group" and type the desired name for the new Attribute Group.

Once you type the desired name, click on Apply or press Enter on your keyboard to save.

![](.gitbook/assets/image\(1153\).png)

&#x20;

A description can be added under the Description box.

Click on Save to make these changes.

![](.gitbook/assets/image\(640\).png)

&#x20;

The new Attribute Group has been created.

![](.gitbook/assets/image\(676\).png)

&#x20;

### Add Attributes to Attribute Groups from Attributes Groups Tab

In the Attribute Groups tab, select an Attribute Group and click on Edit to make changes to the Attribute Group.

![](.gitbook/assets/image\(677\).png)

&#x20;

Now that the details are displayed, click on the arrow on the right to show the +Add and -Remove Buttons and the Attributes stored in the Attribute Group.

Click on +Add.

![](.gitbook/assets/image\(643\).png)

&#x20;

Now select the attributes to be added by typing in the names of the attributes in "Click or start typing box" or

by finding the attributes using the arrows on the bottom of the page.&#x20;

Click on the checkboxes and then press +Add.&#x20;

![](.gitbook/assets/image\(624\).png)

&#x20;

All attributes are added to the Group when clicking on Save.

![](.gitbook/assets/image\(623\).png)

###

Clicking on the Attributes will take you to the Attribute Properties.

![](.gitbook/assets/image\(626\).png)

###

### Add Attributes to Attribute Groups From Classification Tab

Alternatively, attributes can be added from the Classification tab.

Select a category, and checkmark the attributes to be added.

Click on operations, and select Add to Group...

&#x20;

![](.gitbook/assets/image\(678\).png)

&#x20;

Now select the Attribute Group where the chosen attributes will be added.&#x20;

Click on +Add.

![](.gitbook/assets/image\(633\).png)

&#x20;

Go back to the Attribute Groups tab and select the Attribute Group to display the added attributes.

&#x20;

![](.gitbook/assets/image\(679\).png)

&#x20;

### Add Attributes to Attribute Groups From Attributes Tab

Alternatively, attributes can be added from the Attributes tab.

Select an attribute and click on Edit.

&#x20;

![](.gitbook/assets/image\(680\).png)

&#x20;

Click on the Arrow next to Attribute Groups.

Click on +Add.

![](.gitbook/assets/image\(681\).png)

&#x20;

Now select the Attribute Group where the selected attribute will be added.&#x20;

Click on +Add.

![](.gitbook/assets/image\(647\).png)

&#x20;

The attribute has been added to the Attribute Group.

Click on Save.

![](.gitbook/assets/image\(682\).png)

&#x20;

Click on Save to confirm.

![](.gitbook/assets/image\(649\).png)

&#x20;

Now the selected Attribute Group is shown under Attribute Groups.

If you click on it, it will take you to the Attribute Group tab and display all details.

![](.gitbook/assets/image\(683\).png)

&#x20;

### Delete Attributes from Attribute Groups Tab

&#x20;

Go to the Attribute Groups tab and click on the Attribute Group to be deleted.

![](.gitbook/assets/image\(685\).png)

&#x20;

In the Attribute Group details view, click on the arrow next to Attributes in Group to display all attributes stored in the Attribute Group.

By click on the attributes, attribute details can be displayed.

Click on Delete to erase attribute Group.

![](.gitbook/assets/image\(631\).png)

&#x20;

Click on Delete to confirm.

![](.gitbook/assets/image\(636\).png)

&#x20;

### Delete Attributes From Classification Tab

&#x20;

Follow the guidelines from "Add Attributes to Attribute Groups From Classification Tab" and select -Delete instead of +Add.

&#x20;

### Delete Attributes to Attribute Groups From Attributes Tab

&#x20;

Follow the guidelines from "Add Attributes to Attribute Groups From Attributes Tab" and select -Delete instead of +Add.
