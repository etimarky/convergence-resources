**Description Generator – Using Abbreviations** 


 


The category processing action **Description Generator** allows 
the user to utilize the Abbreviation Dictionary.  This gives the user the 
option to update any field with the abbreviations from the Abbreviation 
Dictionary as long as they are in there.  


 


Note:  This is case sensitive.  The Value must match the case of 
the Value in the Abbreviation Dictionary.


 


1.      
Set up the Template as desired by selecting the Target attribute.


2.      
Select the Target Attribute to update


3.      
Insert the Abbreviation Value (This will take the abbreviation if 
exists and update the attribute.)


4.      
Save the Template


5.      
Click the Run radio button


6.      
Verify the Correct Template is selected, the correct batch is 
selected and the desired status


7.      
Click the Run button


 


            ![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_Abbr_files/image001.jpg)




 


 


 The abbreviations will now display for those items.
