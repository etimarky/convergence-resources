# Attribute Filtering


Navigate to SmartFind and drill into a category where you would like to view and access parts. 


 


Click on the Funnel icon highlighted below and this will take you to SmartFind Advanced Filtering. 


 


![](assets/image_70.png)




 


 


The Advanced Filtering Menu comes up and you can choose, System Attribute Filters or Attribute Filters. Let's click the Attribute Filters button on the top.


 


Click the plus sign to add an attribute filter. 


 


![](assets/image_71.png)




 


 


Click the drop down menu to choose which operator you would like to use. 


Since I am filtering on an attribute with a String data type I will be using the "is" operator because you cannot use logical operators on string values. 


 


If you are filtering on a numeric attribute you can use the logical operators that are offered in SmartFind's advanced filtering. 


![](assets/image_72.png)




 


Type in what you would like to filter by in the box to the right of the operator box, and click apply. 


 


![](assets/image_73.png)




 


Now the list of items will be filtered by if the finish is cadmium plated. The current active filters show up on the top of the screen in the "chip" icons, highlighted below. 


 


![](assets/image_74.png)




 


 


You can add as many filter "chips" as you would like and they will add on to one another to refine your filtering.
