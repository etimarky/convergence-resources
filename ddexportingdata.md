**Exporting 
Data**



The 
Data Developer allows users to export data to either Excel (xls) or Text (csv) 
Format.  Any data retrieved in Data Developer can be exported.  


 


The 
user can choose to export all the items or they can choose to select the number 
of items to export by using the Shift key or the Control key to highlight 
multiple rows.  Once they have the data you wish to export displayed in the 
main grid, choose your report format from the Reports menu.


 


Select 
the rows you wish to export, and choose the Export type from the Reports Menu.


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Exporting-Data_files/image001.jpg)




**Example**: Step 1 - Exporting 
2 rows of data as a text file.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Exporting-Data_files/image002.jpg)




**Example**: Step 2 - Exporting 
data as text (csv).


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Exporting-Data_files/image003.jpg)




**Example**: Step 2 - Exporting 
data as excel (xls).


 


 


After 
choosing the export type, the user is prompted to save the file to a particular 
location.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Exporting-Data_files/image004.jpg)




**Example**: Saving two rows of 
data as a text (csv) file as c:\reports\text\two\_rows.txt


 


 


For 
both exporting options, any columns that are hidden in the grid will not get 
exported.  The data will be exported in the same order the attributes are in 
Data Developer.
