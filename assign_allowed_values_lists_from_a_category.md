# Assign Allowed Values Lists(AVL) from a Category


 


In classification, locate the desired category.


Select the desired category where the AVL will be added.


![](assets/image(888).png)




 


Select the attribute where the created AVL will be assigned.


Under Details, click on Assign.


![](assets/image(889).png)




 


Find an existing AVL to assign or create a new one.


Optionally, a new AVL can be created by typing on "Add New AVL"; thus, the previous step can be skipped.
