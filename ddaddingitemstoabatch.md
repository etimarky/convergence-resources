**Adding 
Items to a Batch**



A working batch is a collection of items that already exist in the 
database.  It is different from a load batch in that the items already exist in 
the database.  If the user deletes a working batch, he/she will not delete the 
underlying items in the working batch, only the working batch.  


 


**Create a Working Batch**


To add items to a working batch in Data Developer, select the 
item(s) using the filtering capability, right click, and choose “Add Items to 
Working Batch…”


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Adding-Items-to-a-Batch_files/image001.jpg)




**Example**: Adding three items 
to a working batch.


 


After 
selecting the “Add Items to Working Batch…”, the user will be prompted for the 
batch name. The user can either type in a new name for the batch, or select an 
existing batch name from the dropdown list.  To remove all existing items from 
the batch prior to adding the new items, select the Clear Existing Items box.


 


**Note**:  It is a good idea 
to end your batch name with the prefix \_working to identify it as a working 
batch.

**Retrieve 
an Existing Working Batch**


Select 
the arrow to the right of the Batch field at the top of the window.  Select the 
Working Batch name.  Go to the category containing items you want to retrieve 
from the batch.


 


**Add 
Items to an Existing Batch**


After 
selecting the “Add Items to Working Batch…”, the user will be prompted for the 
batch name. The user can select an existing batch name from the dropdown list.  
To remove all existing items from the batch prior to adding the new items, 
select the Clear Existing Items box.


 


**Delete 
Items from a Working Batch**


Retrieve the contents of the target Working Batch. Right 
click on the first column of the Data Developer.  Select **Manage Working 
Batches...** The **Manage Existing Working Batches** pop-up window 
will be displayed. Select the batch from which items will be deleted from the list of 
working batches. Select the item or items to be deleted in the bottom window, and 
select the **Delete Item(s)** button.  A **Deleting Working Batch Items** confirmation window is 
displayed. Select **Yes** to confirm the deletion.  Select **F5**to refresh the display and confirm that the item has been deleted.


**Delete a Working Batch**


Right click on the first column of the Data Developer.  Select **Manage 
Working Batches...** The **Manage Existing Working Batches** pop-up 
window will be displayed.  Select the batch to be deleted from the list of working batches.  Select 
the **Delete Batch** button.  A **Deleting Working Batch** confirmation window is displayed. 
Select **Yes** to confirm the deletion.
