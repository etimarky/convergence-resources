



DDFilters - Design For Retrieval (DFR) Help




**Filters**



All 
of the data displayed in Data Developer is based off of the filters in the 
upper left corner.


There 
are filters for Batch, User, and Status.


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Filters_files/image001.jpg)




**Example**: Filters set to 
return all data.


 


Adjust 
the filters to return the desired results.  For example, to only see the items 
in the “new batch name” batch, adjust your filters to select only that batch:


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Filters_files/image002.jpg)




 


You 
will notice only the items in that batch are displayed in the category tree.


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Filters_files/image003.jpg)




**Example**: Filtering on a 
batch name.


 


Similarly, 
the user can filter on items with a specific status, or owner.


 


To 
toggle between the selected Filter and the “All” mode, double click the text 
next to the dropdown.  For example, if “new batch name” is currently selected, 
you can toggle to “--ALL BATCHES--” by double clicking “Batch”.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Filters_files/image004.jpg)




**Example**: To toggle “new 
batch name” to “--ALL BATCHES--”, double click the label next to the dropdown.


 


Similarly, 
you can double click the “User” or “Status” label, to toggle between the 
selected value, and the “ALL” value.


 


When 
the user exits DFR, the values they had selected are saved, are restored the 
next time the user opens Data Developer.
