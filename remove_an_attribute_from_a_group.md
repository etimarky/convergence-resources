**Remove an Attribute from a Group**


 


To Remove an attribute from a group:


 


- Select the attribute and press the [Edit](javascript:popuplink('hs2120.htm')) button.
- Select the **Group** tab.

Note that the Name and Description Table lists all groups to which the attribute is assigned. 


- Place the cursor on the group from which the attribute is to be removed.
- Press the **Remove** button. The group Name and Description is removed from the table.
- Press the **Save** button at the top of the Attribute Manager screen.
