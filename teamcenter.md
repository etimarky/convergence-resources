# Teamcenter Integration


 


DFR has the ability to export data in the correct format to be brought over into a Teamcenter Environment. 


However, in order to make sure the information comes over in the correct format, there are a number of things you must make sure you set up first.


These pages will help educate you on what steps you will need to take in order to make sure the information you are bringing into your Teamcenter environment is clean and matches the information you have in DFR.
