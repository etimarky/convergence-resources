# Search for Parts


SmartFind is a search engine designed to locate specific parts, categories, or even keywords that relate to parts.


SmartFind will locate the desired category or part so the user is then able to edit, reclassify, assign, approve, or reject parts. Searching for a Part can be done in multiple ways.
