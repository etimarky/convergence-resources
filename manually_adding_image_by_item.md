# Manually Adding Image by Item



## Manually Adding Image by Item &#x20;

&#x20;

Permissions are needed to add an image to a part. &#x20;

Now images can be added to an item for display in Convergence PIM.

Go to SmartFind and find a part where the attribute "Image" was added.

&#x20;

![](.gitbook/assets/image\(735\)_1.png)

&#x20;

Click on Edit to see the Item Details of an item, now the option of adding an image is available.

Click on +Add to load an image.

&#x20;

![](.gitbook/assets/image\(816\).png)

&#x20;

Click on the document to be displayed.

&#x20;

![](.gitbook/assets/image\(817\).png)

&#x20;

Now the image is displayed under the selected item.

&#x20;

![](.gitbook/assets/image\(818\).png)

&#x20;

In Edit, if you would like to add more images to the same item, hover over the image and select "+Add".

If you would like to delete an image from an item, hover over the image and select "-Remove".

&#x20;

![](.gitbook/assets/image\(819\).png)

&#x20;

Now multiple images are displayed under a part. These can be modified individually by clicking on the arrows.&#x20;

&#x20;

![](.gitbook/assets/image\(820\).png)
