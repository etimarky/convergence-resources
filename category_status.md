



Category\_Status - Design For Retrieval (DFR) Help




**Category Status**


 


 


- **New** - indicates a 
 newly created node that has had no status changes
- **Pending** - indicates that 
 a node has been submitted by the developer and is waiting for approval by 
 the approver
- **Approved** - 
 indicates that a node has been approved by the approver
- **Rejected** - indicates that 
 a submitted and approved category has been rejected by the approver. This 
 may indicate that circumstances have changed regarding this category, or 
 that it is obsolete

Additional User Defined statuses can be entered 
in the Status Maintenance Administrative tool in DFR Online. These statuses 
allow the user to define various statuses through which a Category may reside 
between New and Pending.
