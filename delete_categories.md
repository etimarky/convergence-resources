# Delete Categories


Deleting a category or group of categories removes them from the structure indefinitely. 


Parts must be removed from the category before it can be deleted
 




<iframe src="https://www.youtube.com/embed/ZBiWu7qD8Ek/?wmode=transparent" width="560" height="315" frameborder="0" allowfullscreen></iframe>


 


In classification, locate the desired category using the category tree.


![](assets/image(1177).png)

  

 


Select the three dots on the right side of the category name.


Select " Delete"


Select " Category"  

 


![](assets/image(1178).png)

![](assets/image(1179).png)




 


If the user aims to delete a parent and child categories, simply select just the parent and select delete. 


A window will be prompted informing the user of the category/categories that will be deleted. 


Select OK.


 


![](assets/image(1180).png)




The category has been successfully deleted.
