# Delete Categories

Delete\_Categories - Design For Retrieval (DFR) Help

## Delete Categories

Deleting a category or group of categories removes them from the structure indefinitely.&#x20;

Parts must be removed from the category before it can be deleted &#x20;

&#x20;

In classification, locate the desired category using the category tree.

![](.gitbook/assets/image\(1177\).png)

&#x20;

Select the three dots on the right side of the category name.

Select " Delete"

Select " Category"

&#x20;

![](.gitbook/assets/image\(1178\).png)

![](.gitbook/assets/image\(1179\).png)

&#x20;

If the user aims to delete a parent and child categories, simply select just the parent and select delete.&#x20;

A window will be prompted informing the user of the category/categories that will be deleted.&#x20;

Select OK.

&#x20;

![](.gitbook/assets/image\(1180\).png)

The category has been successfully deleted.
