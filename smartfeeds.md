# SmartFeeds


**SmartFeeds** is the CDS PIM module where you can manage import feeds. There are two types of supported feeds:


- Import Feed
- Deliverable Import Feed


 


**Import Feeds**


- Used to automate importing data to update items
- Triggers are defined by time
- Example: a feed can be configured to watch an SFTP folder and automatically import a file added there


 


**Deliverable Import Feeds**


- Used to automate updating items using the new AutoEnrich task type in SmartTrack deliverables
- Triggers are defined by item status
- Example: an item going through the process flow steps in a SmartTrack deliverable reaches an "auto enrich" step; the item is automatically updated by an external source and pushed to the next step in the deliverable
