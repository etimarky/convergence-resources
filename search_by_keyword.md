# Search by Keyword


Searching for parts can be done by Keyword Search.


The Keyword Search option allows the user to generate results using specific keywords of the part, description or values under any of the part's attributes.


 


To search for a part by Keyword, select Keyword Search from the drop-down menu.


![](assets/image(332).png)




 


Type in any keyword that would aid in locating a part and select Search. 


In this case, results will appear displaying any part that has the word "bolt" in any attribute, description or category. 


![](assets/image(560).png)




 


All the items that contain the word "bolt" in any of their attributes are displayed.


On the left side of the page, you can see the categories where those items are located.


Clicking on any of the item numbers will take you the main part and display all its details. 


![](assets/image(562).png)
