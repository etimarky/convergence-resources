## Moving Modules – Drag and 
Drop


If a user right clicks on a Module header row, that user can 
move that module, and all of its variants, to another position on the 
roadmap.  After right clicking, the user drags that header row up or down 
to the new position on the road map.  A user should notice when you select 
a header row, a hint appears reminding users to use right click to drag 
modules.


 


Once a user finalizes their module row configuration, the 
new configuration can be saved and can also be printed.



**![](IM-Filter_move_Copy_files/image003.jpg)**


**Example of how to move Modules in a roadmap**




## Copy Into Next Year


If a user wants to copy data from a variant to the next 
variant in sequence, there is now an option called Copy into next year.  
If the user selects Details (see below) there is an option to copy this 
data.  A user can only copy data to the next variant (one at a time). An 
alternate option is to click the variant year, and press the *Insert* key 
on the keyboard.


 



**![](IM-Filter_move_Copy_files/image004.jpg)**


**Example of Copying Variant Data to the Next Variant**



**Deleting 
Variant Years**


If a user wants to delete a variant, they can right click on 
a variant, and choose *Delete item-yr*.  An alternative, is click on 
variant, and press the *Delete* key on the keyboard. Only users with the 
Super User role (see section below) can delete variants in the base year. 


![](IM-Filter_move_Copy_files/image005.jpg)




 


**Multi-selecting 
of Variants**


There are two different ways to multi-select variants.  
Holding the *Control* key while clicking on a variant allows for multiple 
variants to be selected. The other way to multi-select is to left click and 
drag the mouse.  All variants within the region chosen will be selected.


 


![](IM-Filter_move_Copy_files/image006.jpg)




**Example of multi-selecting variants**



Once multiple variants are selected, they can be deleted by 
pressing the *Delete* key, or copied into the next year by pressing the *Insert* 
key.


 


![](IM-Filter_move_Copy_files/image007.jpg)




**Example of deleting multiple variants**


 


**Deleting 
Individual Replacements**


To delete an individual replacement, right click on the blue 
line, and choose *Delete Line.*


**![](IM-Filter_move_Copy_files/image008.png)**


**Example of deleting a replacement**



 


**Read 
Only Data for Base Year**


The Item Mapping tool does not allow normal users to edit 
variant year data for the base year by setting certain fields to read 
only.  A user will notice a gray background that says “Read-only data” in 
the baseline column, as shown below.  Certain right click menu fields have 
been grayed out for the read only data to protect it from any editing.


 


![Description: C:\Code\Trunk\DFR Help\html\IM-Filter_Move_Copy_files\image005.jpg](IM-Filter_move_Copy_files/image005.jpg)




**Example of read only data for the base year 2009**


 


**Super 
User Role**


A user with the *Super User* role is able to make 
changes to the base year, as well as any other variant year regardless of its 
owner group.
