## Set Attributes as Key, Required, DNA, and Read Only

&#x20;

&#x20;

First navigate to SmartClass and click the orange "Category Tree" button in the top left.&#x20;

&#x20;

&#x20;

![](.gitbook/assets/image_36.png)

&#x20;

Now you can click the orange "thumbtack" button to pin the tree on the page.&#x20;

&#x20;

Then drill down to the category you would like to change the attribute features.&#x20;

![](.gitbook/assets/image_37.png)

&#x20;

&#x20;

Find the attribute that you would like to edit and click on the arrow button on the far right of the attribute shown below.&#x20;

![](.gitbook/assets/image_38.png)

&#x20;

Now in this window click the blue Edit button

![](.gitbook/assets/image_39.png)

&#x20;

You can now click the greyed out icons to enable the feature or click the colored icons to disable the feature.&#x20;

&#x20;

The circle with the exclamation inside is Required: This means the attribute is required to have a value.&#x20;

The eye icon is Read Only: This means no one except for system admin can edit this attribute.

The circle with the cross through it is DNA (Does Not Apply): DNA means that this attribute does not apply to this specific category.&#x20;

The key icon is Key: This means that the attribute is important (this is usually used for filtering purposes).

&#x20;

![](.gitbook/assets/image_40.png)

&#x20;

To finish editing your attribute you can click the green save button to save your progress.&#x20;
