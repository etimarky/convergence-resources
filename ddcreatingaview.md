**Creating 
a View**



Once 
the user has configured a category to their liking, (including custom filters, 
hiding/aligning columns) it can be saved as a view.  To create a view, select 
the “Views…” button:


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Creating-a-View_files/image001.jpg)




**Example**: Clicking the Views 
button.


 


After 
clicking Views, the user is presented with a dialog.  Next to the *Save as* 
field, enter the name of the view.  You have the choice of saving the view as 
public or private.  Only you can access the private views, whereas public views 
are shared with all users.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Creating-a-View_files/image002.jpg)




**Example**: Saving a Private 
View with a name of “MyView”.


 


*Note*: After clicking 
“Save as”, the view will appear in the list of views.
