**Item 
Mapping Configuration Options**



The following items will be configurable.


- All of the MM options below in DFR Options Manager are 
 specific to Item Mapping
- MM\_Base Year – the year the map starts from
- MM\_Last Year –the last year maps go to
- The other MM options are specific to the data model
- FTP settings for category attachments
- Owner Group settings

**Example of Owner Group Settings in the Options Manager in 
DFR**


 


The owner groups listed above in the options table will 
always be listed first before any other owner groups that may exist in the 
Whirlpool database.  This order can be changed and new owner groups can be 
added.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/IM-Configuration_Options_files/image004.jpg)




**Example of Owner Group to Product group mappings in DFR**



If the name of the Owner Group does not map the name of the 
Product Group, you can set the mapping here. In the example above the Owner 
Group *G273* is the same as the Product Group *dish*.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/IM-Configuration_Options_files/image005.jpg)




**Example of FTP settings in the Options Manager in DFR**



When a roadmap is saved it generates a roadmap file that 
gets stored in an FTP directory.  A link to this file is created and assigned 
to a module category via category attachments.  When a user searches on a 
category attachment in SmartFind, they can open and view the latest roadmap 
file.  The configuration settings above are for the FTP location for the 
roadmaps. 



**![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/IM-Configuration_Options_files/image006.jpg)**


**Example of Roadmap category attachment in DFR’s 
Classification Manager**



Users can click on this URL link.  It will show the latest 
roadmap that has been saved for a particular module type.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/IM-Configuration_Options_files/image007.jpg)




**Example of where the roadmap category attachments are 
saved in DFR**



The category attachments are saved under the attachments 
category in DFR, like other category attachments.

**Example of where roadmap attachments appear in SmartFind**
