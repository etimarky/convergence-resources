# Add Values Manually & Suggested Values


This feature allows you to add values to an AVL:


- Values can be added manually
- Values can be added if there are existing values under the attribute where the AVL was created; this feature automatically compiles all existing values and allows you to add them in one step.


 


Go to the Allowed Values page to select an attribute.


![](assets/image(952).png)




  

Under Values, select Edit Values.


![](assets/image(953).png)




 


In order to add all values under the attribute where the AVL was created, click on **Add Suggested** to add all existing values.


Optionally, you can add new values by simply typing a value in the **Add New Values** box.; this option allows you to add new values to the AVL.


![](assets/image(834).png)




 


Now, click on Add All Suggested to add all values or select the checkboxes to add individual values.


![](assets/image(836).png)




 


Click on Confirm.
