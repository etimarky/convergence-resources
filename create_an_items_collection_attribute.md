**Create an Items Collection Attribute**


 


 


A Collection Attribute is a special attribute that can have many values associated with it. The values assigned to the collection attribute are called "members". To learn more about membership in an item collection, review the Assign Members to an Item Collection topic.


 


**To create a Collection Attribute:**


 


- From the Attribute Manager window, select **Add**

 


- Enter the **Name** and **Description** of the collection attribute

 


- Select the **Save** button to save the attribute
