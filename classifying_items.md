**Viewing 
Attribute Attachments**


 


The Data Developer allows users to view and open attribute 
attachments.  These Attachments are added in Attribute Manager.  (See Attribute 
Manager Help)  Attachments cannot be edited, deleted or updated from here, only 
viewed.


 


**Viewing Attribute Attachments:**


 


·         
Retrieve the item or items you want to view from Data Developer.


·         
Select the column title of the attribute you would like to preview.


·         
At the top of the right click menu will display Attribute 
Information.  Select this.


  
![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Attribute-Attachments_files/image001.jpg)




Right 
click option menu


·         
In the Attribute Information dialog box, you will see the list of 
Attribute Attachments.


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Attribute-Attachments_files/image002.jpg)




**List of 
Attribute Attachments**



·         
Double click on any attachment in the list to preview.
