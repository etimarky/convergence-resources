**Status of Categories**


 


The color of the folder to the left of the 
category or terminal node name is a status indicator designating the status of 
the node. The colors for your installation are defined in the Status Maintenance 
tool in the Administration module within DFR Online. 


 


 


**To change the status of a category:**


 


- **Select** the category from 
 the classification tree

 


- Right 
 click on the category or select the Actions pull down menu

 


The status and the color of the category icon 
will change as listed above.
