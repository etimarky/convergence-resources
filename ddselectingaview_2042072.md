**Selecting 
a View**



To 
select a view, navigate to the category of interest or select the Root node, 
and click the “Views…”  Button.




**![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Selecting-a-View_files/image001.jpg)**


**Example**: Clicking the views 
button.


 


After 
clicking the views button, a dialog is presented for the user to select a previously 
saved view.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Selecting-a-View_files/image002.jpg)




**Example**: To load a view, 
select it in the View Manager, and click “Load View”.
