# Changing System Attributes


The ability to change item numbers after initial import can be a useful tool in the event that the original number/name was a placeholder of some kind or there is a change in the item number itself. 


 




<iframe src="https://www.youtube.com/embed/ux1VfV9bu-o/?wmode=transparent" width="560" height="315" frameborder="0" allowfullscreen></iframe>


 


Note: There are permissions that must be in place for this to take place:  

-Ability to edit parts
Locate Data Developer in the ThickClient


Locate the category and then part number that needs to be edited/changed. 


 


Note: only one part number at a time can be changed
Right-Click the row arrow and select " Edit System Attributes"


 


Note:  

The part has to be assigned to the user logged in.  

The part has to be in any status outside of pending or approved.
![](assets/image(949).png)




 


Once the window is opened up, the user is allowed to change such system attributes as:


- Item Number
- Item Description
- Legacy Item Number
- Revision
- Qualifier
- Release Date


These options are usually grayed out when in Data Developer, and cannot be changed by regular users. ( See Permissions at the top of the page.)


 


![](assets/image(951).png)




 


Select Save. 


The Item Number has been updated.
