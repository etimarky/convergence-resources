## Changing System Attributes

The ability to change item numbers after initial import can be a useful tool in the event that the original number/name was a placeholder of some kind or there is a change in the item number itself.&#x20;

&#x20;

&#x20;

Note: There are permissions that must be in place for this to take place:

-Ability to edit parts Locate Data Developer in the ThickClient

Locate the category and then part number that needs to be edited/changed.&#x20;

&#x20;

Note: only one part number at a time can be changed Right-Click the row arrow and select " Edit System Attributes"

&#x20;

Note:

The part has to be assigned to the user logged in.

The part has to be in any status outside of pending or approved. ![](.gitbook/assets/image\(949\).png)

&#x20;

Once the window is opened up, the user is allowed to change such system attributes as:

* Item Number
* Item Description
* Legacy Item Number
* Revision
* Qualifier
* Release Date

These options are usually grayed out when in Data Developer, and cannot be changed by regular users. ( See Permissions at the top of the page.)

&#x20;

![](.gitbook/assets/image\(951\).png)

&#x20;

Select Save.&#x20;

The Item Number has been updated.
