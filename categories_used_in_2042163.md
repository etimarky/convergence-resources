**Finding Categories in which an Attribute is Used**


 


The **Categories used in** tab appears in the lower half of the the right column of the Attribute Manager window. This tab displays the category path, from the root category down, for all categories where an attribute has been assigned.


 


**To locate all the categories where an attribute has been assigned:**


 


- Select the attribute from the Master Attribute List

 


- Select the **Categories used in** tab

 


The Category Paths will be listed to each category where the attribute has been assigned.
