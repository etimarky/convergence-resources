# Edit Category Details


Editing categories is a quick and simple process done in SmartClass through the Classification module. 


 


 




<iframe src="https://www.youtube.com/embed/_RbWUlVqrbk/?wmode=transparent" width="560" height="315" frameborder="0" allowfullscreen></iframe>


 


 


In classification, locate the desired category using the category tree.


Select the orange Edit icon in the top right of the category.


![](assets/image(1032).png)




 


The category can now be edited. 


Many features can be changed, such as: 


- [Category Image](#)
- Category Name
- Category Description
- Category Status
- Category Authorization
- Category Attribute Inheritance
- Category Attachments ( This is not currently available in Version 10. This will be available in a later release.)


![](assets/image(1031).png)




 


Make the desired changes and select the save button in the top right.
