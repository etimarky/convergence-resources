## Edit Category Details

Editing categories is a quick and simple process done in SmartClass through the Classification module.&#x20;

&#x20;

&#x20;

&#x20;

&#x20;

In classification, locate the desired category using the category tree.

Select the orange Edit icon in the top right of the category.

![](.gitbook/assets/image\(1032\).png)

&#x20;

The category can now be edited.&#x20;

Many features can be changed, such as:&#x20;

* [Category Image](edit_category_details.md)
* Category Name
* Category Description
* Category Status
* Category Authorization
* Category Attribute Inheritance
* Category Attachments ( This is not currently available in Version 10. This will be available in a later release.)

![](.gitbook/assets/image\(1031\).png)

&#x20;

Make the desired changes and select the save button in the top right.
