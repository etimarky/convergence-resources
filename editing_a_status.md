# Editing a Status


Editing a status will give the user the option to edit various elements of the status such as the color of the status tag, name, and what data will be flagged under the status such as items, categories, and parts.


 


Permissions need to be set and allowed for users to utilize the catalog manager and status manager.   

These permissions include:   

-Classification Permission: Catagory Permission/ Edit Items  

-Catalog Permission: Access Catalog Status Module
 


Editing a status is easily accomplished by navigating to the **[Status Mangemen](#)t** screen, selecting the status and then " Edit" on the status to the left. 


 


![](assets/image(928).png)




 


Once the edit screen has been opened, the user is given the choice to change any of the information such as the:


- Status Type (Name)
- Which item(s) are being covered with this particular status
- The color of the status tag which is visible in various SmartClass views, identifying the status by color instead of name
- Usage: this will notify the user(s) of the amount of attributes, categories, category attributes or items that currently are under the selected status


 


![](assets/image(929).png)




 


Select Save when complete.
