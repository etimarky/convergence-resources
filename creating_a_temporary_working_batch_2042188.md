**Creating a 
Temporary Working Batch**


 


The 
Data Developer can be used to filter a subset of items and identify them as a 
temporary batch. The temporary batch can be useful for managing items and 
developing data.


 


Working 
batch functions can be accessed from the main grid of the data developer, once 
items have retrieved and selected.


 


**To create a working batch:**


·         
Use the filtering capability of the Data Developer to select a set 
of target items.  


·         
When the desired items are displayed in the grid, select the items 
to be added to the temporary batch.


·         
Right click on the first column on the left of the grid to bring 
up the working batch menu. 


·         
Select the **Add Items to Working Batch…** pull-down menu and 
type in your new working batch name.


·         
Press OK.


 


**To retrieve an existing working batch:**


·         
Select the arrow to the right of the **Batch** field at the top 
of the window. 


·         
Select the working batch name. 


·         
Go to the category containing items you want to retrieve from the 
batch.


 


**To add items to an existing working batch:**


·         
Use the filtering capability of the Data Developer to select a set 
of target items.  


·         
When the desired items are displayed in the grid, select the items 
to be added to an existing temporary batch.


·         
Right click on the first column on the left of the grid to bring 
up the working batch menu. 


·         
Select the **Add Items to Working Batch…** pull-down menu and 
select the working batch you want to add items too.


·         
Press OK.


 


**To delete items from a working batch:**


·         
Retrieve the contents of the target working batch. 


·         
Right click on the first column of the data developer


·         
Select **Manage Working Batches...** The **Manage 
Existing Working Batches** pop-up window will be displayed. 


·         
Select the batch from which items will be deleted from the list of 
working batches. 


·         
Select the item or items to be deleted in the bottom window, and 
select the **Delete Item(s)** button. 


·         
A **Deleting Working Batch Items** confirmation window is 
displayed. Select **Yes** to confirm the deletion. 


·         
Select **F5** to refresh the Item Browser Display and confirm 
that the item has been deleted.


 


**To delete a working batch:**


·         
Right click on the first column of the data developer


·         
Select **Manage Working Batches...** The **Manage 
Existing Working Batches** pop-up window will be displayed. 


·         
Select the batch to be deleted from the list of working batches. 


·         
Select the **Delete Batch** button. 


·         
A **Deleting Working Batch** confirmation window is displayed. 
Select **Yes** to confirm the deletion.
