**Setting 
the Status for an Item**


 


Data Developer allows certain users to change the status of an 
item.  Only those users that are either Catalog Administrators or Item 
Approvers can change the status of items.  An item does not have to be assigned 
to a user to change its status.  


 


To set the status of an item in Data Developer, select the item 
(or items) by clicking on the number to the left.  To select multiple items in 
consecutive order, use your shift key and to select multiple items not in 
order, use the Ctrl key.  Once selected, right click and select “Set Status” 
and select the status from the list.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Setting-the-Status-for-an-Item_files/image001.gif)




 


**Example**: Setting an item’s status to “Pending”.


 


The status of an Item can be set to Pending, Approved, or 
any User Defined status.  Once an item is set to Pending or Approved that item 
can no longer be edited.  (This means that data in any cell can no longer be 
changed or updated.)  You will notice the item is no longer in darker print.  A 
user can set the status back from Approved to Pending or back to any User 
Defined if they choose.  Once an item has been soft deleted (status = 
Obsolete), its status can no longer be changed.  


 


In order to set an item to Approved, the user must have the 
role Item Approver from User Manager.  If the user does not have this role, the 
Approved role will be grayed out on the toolbar.
