**Item Duplicates**


 


**Application 
Overview**


The Duplicate Finder module will 
help find duplicate items in the database based on category and attribute criteria 
specified by the user.  Matches are based on exact values of the attributes 
selected.  Any DFR user has access to this application.  Duplicate analysis can 
be run across multiple categories.



**Initial Log-in**


Select the Duplicate Finder icon 
to open the module.  Then select the type of analysis to conduct from the drop 
down menu.  Start the duplicate search by selecting Start Duplicate Search.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/Item%20Duplicates%20Help%20File_files/image001.jpg)




**Category and 
Attribute Selection**


Select the category from where to 
conduct a duplicate analysis.  It does not have to be a terminal node category, 
an analysis can also be run at a higher level category which will include all 
child categories in the results – also known as an analysis across multiple 
categories.


 


Once the category is selected, 
the category attributes are displayed.  Select which attributes Duplicate 
Finder will analyze.  Duplicate Finder analysis tool will find items that have 
duplicate values for the attributes selected.  The more attributes that are 
selected the higher probability of finding actual duplicate parts, however the 
number of duplicate results will decrease.  Assuming that is the objective of the 
analysis.  In contrast, selecting only want a few attributes can be utilized to 
find similar parts based on a few similar characteristics.


 


Once a category and attributes 
have been selected, run the analysis by selecting Find Duplicates.


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/Item%20Duplicates%20Help%20File_files/image002.jpg)




 


At any point, you can select the 
back arrow to start over.  If you don’t go back to the very first screen, the 
application will remember your previous duplicate analysis selections for 
multiple categories.  If you go back to the very beginning, all previous 
analysis selections will be reset.


 


**View Results**


The results show the duplicates 
sets found based on your selection criteria. You can view additional data on 
each group of results to conduct further analysis.  


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/Item%20Duplicates%20Help%20File_files/image003.jpg)
