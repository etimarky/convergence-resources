**How 
to Fix Summary Reports**



We 
have come across this issue before from an actual bug from within Microsoft 
Windows.  If you have a problem printing a summary report, you may want to 
follow these steps.


 


1)      
Open 
Control Panel


2)      
In 
Control Panel open Regional and Language Options


3)      
Under 
Standards and Formats please remember the selection in the dropdown.


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/IM-Fix_Summary_Reports_files/image001.jpg)

 


1)      
Select 
a different format like this and click OK.


2)      
Re-open 
Regional and Language Options again.


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/IM-Fix_Summary_Reports_files/image002.jpg)




3)      
Select 
the original Language that was in the computer before we adjusted it and now 
click OK.


4)      
You 
may or may not need to restart you computer, if the computer does ask to be 
restarted please restart before trying to run the report again.
