**Creating 
and Editing Attributes**


 


An attribute may be added, modified or deleted using the [**Add**](javascript:popuplink('hs2110.htm'))**, [Edit](javascript:popuplink('hs2120.htm'))**or**[Delete](javascript:popuplink('hs2130.htm'))** button at 
the top of the center column. When you select Add or Edit, the action options 
will change to [**Save**](javascript:popuplink('hs2140.htm')) or [**Cancel**](javascript:popuplink('hs2150.htm'))**.**


 


To add attributes, the user must have the role of Attribute 
Developer or Attribute Authorizer.  Both of these roles can be granted in the 
User Maintenance in the Administration module in DFR Online. 


 


**To add an 
attribute, you need the following information:**


 


·                          
The [**Name**](javascript:popuplink('hs2160.htm')) and [**Description**](javascript:popuplink('hs2170.htm')) for the 
new attribute


·                          
The [**Data Type**](javascript:popuplink('hs2200.htm')) of the 
attribute


 


**Add**


Click on the Add Button.  Type the name of the attribute in the 
Name box.  A description is required.  


 


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/AM-Add_Attributes_files/image001.jpg)




  
  



Select the data *Type* for the attribute.  If applicable, 
select a length, UOM type and UOM.  


 


The Properties tab displays a list of properties that apply to 
this attribute.  (See below.)  This list of properties will update depending on 
the attribute data type selected.  Each data type has its own applicable list 
of properties.  (See Attribute Properties help for the list of properties and 
their description.)


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/AM-Add_Attributes_files/image002.jpg)




 


Save the attribute and it will appear in the list.  Clicking the 
Cancel button will cancel the Add or Edit.  


 


If the user wishes to cancel the entry, simply click the Cancel 
button.  If the user wishes to save the entry, click the Save button.  The 
attribute will now appear in the Master Attribute list.  


 


 


**Edit an Attribute**


  
 
To edit an attribute, select an attribute from the list of Master Attributes.  
Use the Filter options to find the attribute.  


 


Using the wildcard (%), the user can search for Words in Name or 
Words in Description.  The user can also search by Data Type, Unit Type, Group 
or Status.  If the user wants to view all attributes that start with a certain 
letter of the alphabet, put the cursor in the Master Attribute list and type 
the letter from the keyboard.  The list will automatically scroll to the top of 
the list for the letter.  For example, if you type an “L”, it will scroll to 
the first attribute in the list that begins with an “L”.  The Edit button will 
display as active as long as the attribute is not approved.  


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/AM-Add_Attributes_files/image003.jpg)




 


If the attribute is approved, right click, then select one of the 
User Defined statuses on the Rework sub-menu. Setting the status to a Custom 
Defined status will allow the attribute to be edited.  


 


![](AM-Add_Attributes_files/image004.gif)




 


 


**Delete**


To Delete an attribute, select the attribute and click the Delete 
button.  If the Delete button is grayed out when the attribute has been 
selected, this means that the attribute is being referenced in categories.  The 
user must go to the categories the attribute has been assigned and remove them 
from every category.  Once this is complete, the Delete button will become 
active.  Note:  The Categories used in tab will display all categories that 
attribute has been assigned.
