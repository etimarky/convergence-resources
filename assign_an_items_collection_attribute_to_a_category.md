**Assign an Items Collection Attribute to a Category**


 


An Items Collection attribute is treated like any attribute. 


 


**Category Developer rights are required to add attributes to categories.**


 


**To assign an attribute to a category:**


 


- From the left side of the Classification Manager main window, select a node in the classification tree. Note that the right column of the window displays a detailed description of the selected category.

 


- Right click on the category to which attributes will be added or select the Category drop down from the menu bar at the top of the window. The Category Maintenance menu is displayed.

 


- Select **Edit Attributes…**

 


The **Attribute Mapping** window is displayed. From the Attribute Mapping window, the user may add attributes to or delete attributes from a category.


 


- The Master Attribute List is displayed in alphabetical order in the **Master Attributes** column. Use the **scroll bar** on the right to move up and down through the list of attributes to find the attribute to be assigned.

 


- Select the attribute to be assigned and press the **Assign To Category** button. Note that the attribute has been added to the **Attributes Assigned to Category** column. This column lists any attributes currently assigned to the selected category.

 


The [Attribute Filter](javascript:popuplink('hs2400.htm')) may be used to display a shorter, more targeted list of attributes; For instance, filtering by a data type ot Items will result in a subset of all collection attributes.


 


- When the attribute assignment is complete, select the **Update** button at the lower left corner of the window.  The Attribute Mapping window will close, displaying the Classification Manager window. Note that the new attribute has been added to the attributes list in the right column.
