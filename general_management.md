# General Management

General\_Management - Design For Retrieval (DFR) Help

## General Management

In General Management, the user will be able to add or remove pertinent information such as but not limited to Product Manufacturers, Materials and Fastener Types. These will be critical in creating products on the Teardown home screen.&#x20;

&#x20;

The user must have SmartTeardown permissions in order to view this page. &#x20;

&#x20;

&#x20;

&#x20;

Navigate to SmartTeardown, select "Management" in the top Right, and then "General Management."&#x20;

&#x20;

![](.gitbook/assets/image\(42\).png)

&#x20;

![](.gitbook/assets/image\(43\).png)

&#x20;

Choose the area that needs information added. Each page allows to "add" new.&#x20;

&#x20;

![](.gitbook/assets/image\(44\).png)

&#x20;

&#x20;

![](.gitbook/assets/image\(45\).png)

&#x20;

This will prompt a window which allows the user to fill in information and in some cases, add an image at the same time.&#x20;

&#x20;

![](.gitbook/assets/image\(46\).png)

&#x20;

Users can select entries and edit or delete them as well.&#x20;

Edit is in the top right, as well as delete.&#x20;

&#x20;

![](.gitbook/assets/image\(50\).png)

&#x20;

Entries can also be deleted on the left hand side by hovering over them.&#x20;

&#x20;

![](.gitbook/assets/image\(51\).png)

&#x20;

**Translate**

&#x20;

Translate is also an option on some of the pages.&#x20;

&#x20;

![](.gitbook/assets/image\(47\).png)

The translate feature allows the user to fill in translated words and descriptions that users will see when they change the language in the top right of each SmartTeardown page.&#x20;

&#x20;

&#x20;

**Language and Region**

&#x20;

To choose a different language, navigate to the top right of each page and choose a language in the dropdown.&#x20;

![](.gitbook/assets/image\(48\).png)

&#x20;

Next to the language selection is "Region."&#x20;

&#x20;

Regions are added or removed in General Management as well.&#x20;

When selected, the user will only view products added to that region.

&#x20;
