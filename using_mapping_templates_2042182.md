**Using Mapping 
Templates**


 


To 
make the mapping processes easier when loading items into DFR, common 
column-to-attribute mapping templates can be created and saved in the "**Templates**" 
window in the upper right corner of the screen.  Templates can be created in 
any of the three Item Loader modes.


 


**To map a new load file and save a template:**


 


·         
Follow the steps required to **Load 
a new Batch** using a template:


 


Select the **New**.


 


**Use browse button to** select load file**.** Type in 
a **batch name.**


 


Press the **down** **arrow** next to the 
Qualifier Type field and select the Qualifier Type for the batch.


 


·         
Press the **Ok** button. The Item Loader window will display 
the attributes of the load file ("Import File Columns") to the left 
and the attributes available in DFR to in the middle (“Product Attributes”).


 




·         
Select an Import File Column Attribute from the left column and 
select the corresponding Product Attribute from the center column.


 




·         
When all the attributes have been mapped, Place the cursor in the **Templates**field in the upper right corner of the Item Loader Window and select **Save As** and enter a name for the 
template and a description. (Descriptions are optional).


 




·         
Press **OK button**.


 


To 
retrieve an existing template for a load file, select the **Choose** button from the **Templates** field. The mapping will 
automatically appear in the right hand column according to the values saved in 
the template.


 




To 
modify an existing template, make changes to your mappings and select **Save** button from the **Templates** field.  This will override past mappings for the 
existing template with your new mapping changes.


 




To 
delete a template from the templates window, right click on a template name and 
select **Delete**.
