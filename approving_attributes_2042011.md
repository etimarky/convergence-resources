**Attribute Display Order**


 


There is a display order tab in the 
classification manager that can change the default order of attributes for data 
developer.  Intially the default order of attributes in the data developer is set 
to alphabetical, this order can be changed using the display order tab.  Only 
category approvers can change display order for a particular category.  Changes 
to the display order do not propogate to other sub-categories, changes only 
occur for that category.


**To 
change the Display Order:**


·         
Select 
the display order tab for the category  
  



·         
Select 
one of the attributes and use the **Up** or **Down** buttons to move the 
attribute or drag an attribute to a new location in the list.  
  



·         
When 
using the **Up** or **Down** buttons multiple attributes can be moved 
together by holding the shift key and selecting the attributes.  Note, the 
attributes being selected must be next to each other in sequential order.  
  



·         
When 
done setting your display order, select **Save**  
  



·         
To 
restore the display order to alphabetical, select the **ABC** button, then 
select **Save**.


 


The 
feature can be configured to manipulate the default display order in SmartFind.
