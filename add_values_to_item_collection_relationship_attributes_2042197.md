**Add Values to Item Collection Relationship Attributes**


 


Adding values to item collection relationship attributes is done with the Item Browser. 


 


Before values can be added to a membership attribute, a membership attribute must exist for the collection attribute and the it must be assigned to the category where the item is located.  


 


**Hint:** Adding large amounts of relationship attribute information should be done by the Item Loader Module.


 


**The item must be assigned to you before you can add items to a collection membership.**


 


**To add values to an Item Collection Attribute:**


 


- Find the item for the collection using the standard search functions.

 


- Once you have located the item, select the **Collections** tab.

 


- Select the item collection from the **Collections to search** pull-down menu.

 


- Select the Search button.  The items will be displayed in the **Detail Items** window.

 


- Select an item row.

 


- Select and move the slider bar to the right exposing the relationship attribute.

 


- Place your cursor in the cell for the relationship attribute and double click.

 


- The cell will open and allow you to enter data.
