**Properties Assigned to Attributes**


 


An attribute property is essentially an attribute on an attribute. 


 


There is a standard set of attribute properties released with the Design for Retrieval software. These are: RelAttribGroup, ConstrainingCategory and Recommended. **RelAttribGroup** and **ConstrainingCategory** are special properties used to manage Item Collections. Refer to the **Item Collections** topic for an explanation on their functionality.


 


The **Recommended** attribute property has a value of True or False. Setting the Recommended property value to true will cause the attribute header and its column of values to be highlighted in the Data Developer. This is used to alert the user that it is recommended that data be entered in the highlighted column.


 


**To view attribute properties:**


- From the Attribute Manager window, select an attribute from the Master Attribute List.
- Select the **Properties** tab in the attribute description window.

Note that all attributes display the entire list of available attribute properties. 


 


**To assign a value to an attribute property:**


- From the Attribute Manager window, select an attribute from the Master Attribute List.
- Press the Edit button.
- Select the **Properties** tab in the attribute description window.
- Place the cursor in the the **Property Value** column for the target property and enter the value.
- Press the **Save** button.
