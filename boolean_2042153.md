**Boolean**


 


Boolean Attributes are typically represented as either yes or no or true or false.  For example if a pump has a manual connection, an attribute can be written as a question such as "Manual Connection?" where the values are true or false. Another way to approach this example is to create an attribute where "manual" is one of the attribute values for an attribute labeled "Connection Type."
