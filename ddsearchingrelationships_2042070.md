**Searching 
Relationships**



Searching 
for Relationship attributes with a specific value is done using the 
Relationship tab.


 


For 
example, to find all items, which contain the relationship “Link Part to 
Document”, which have a relationship attribute with the value “test”, select 
the Relationship from the dropdown, enter your search criteria, and click 
search.



![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Searching-Relationships_files/image001.jpg)




**Example**: Finding all items 
which contain the relationship “Link Part to Document”, this has a relationship 
attribute with the value “test”.


 


At 
this point, if you double click on the cell, you will drill down into the 
details, where you’ll see the value “test”, which you searched for.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Searching-Relationships_files/image002.jpg)




**Example:** Relationship 
Attribute which contains the text “test” which was searched for previously.
