**Define a Group of Attributes about the Relationship**


 


To define attributes about a relationship, the attribute group and the attribute properties functions are used in the Attribute Manager.


 


**The first step is to create a new attribute group in the Attribute Manager:**


 


- Select an Attribute and press the Edit button.

 


- Select the Groups tab in the lower right corner of the Attribute Manager window.

 


- Define the name of the new attribute group in the Attribute Manager by selecting the **Define Group...** button.

 


- In the last row of the spreadsheet window, enter the name and description of the attribute group.

 


**Note:** The attribute group name has to be a single word and cannot have spaces.  Do not enter the create date or modify date.


 


- Press **OK**.

 


**The second step is to assign an attribute to a group:**


 


- Create or select an existing attribute which will be an attribute on the relationship.  An example might be Purpose of Reference.

 


- Select the attribute.

 


- Select the **Edit** button.

 


- Select the **Groups** tab.

 


- Select the name of the group you created in the first step and select the **Add** button.  The relationship group has now been added to the attribute as a group.

 


**The third step is to associate the relationship attribute to an item collection:**


 


Select an item collection attribute (data type of Items with Properties).


 


- Select the **Edit** button.

 


- Select the **Properties** tab.

 


- For the **RelAttribGroup** property, enter the name of the relationship group in the value cell exactly as it is spelled in the attribute group.

 


 


An item collection can have more than one relationship attribute.
