**Managing 
Working Batches**



The 
Manage Working Batches command is good for deleting multiple items from a 
working batch.  To manage working batches, right click on an item, and choose 
“Manage Working Batches…”:


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Managing-Working-Batches_files/image001.jpg)




**Example**: Selecting “Manage 
Working Batches…”


 


After 
selecting, a dialog box will appear, where the user can either delete or 
protect working Batches, as well as remove items.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Managing-Working-Batches_files/image002.jpg)




**Example**: Deleting a working 
batch.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Managing-Working-Batches_files/image003.jpg)




**Example**: Protecting a 
working batch.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Managing-Working-Batches_files/image004.jpg)




**Example**: Removing an item 
from a working batch. 


 


*Hint*: This dialog allows 
the user to delete multiple items from a batch.
