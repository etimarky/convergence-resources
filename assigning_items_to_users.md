**Assigning 
Items to Users**


 


 


Items 
can be assigned to the current user or to other users listed in the DFR catalog 
from the data developer. 


 


The 
user who makes the assignment must have the **Item Assign** role to perform 
this function. This role is set in the User Manager module.


 


Item 
assignment takes place in two steps: First, a user name is set using the **Assign 
Item...** function. 


 


**To set or change assigned user:**


 


·         
Move the cursor to the left most column 
and select the row of items you desire to assign.  Now right click to bring up the assignment 
menu options.  
  
 
Note: if you select the very first cell, top left in data developer, this will 
select all of the items in the grid.


 




·         
Select the **Assign Item...** menu option.


 


·         
Press the down arrow to the right of the **Assign Items To:** 
field and select the desired user name.  
  
 
Note: once you select a user the assignment occurs at that point.  If the user doesn’t match the user you 
currently have selected in the user combo box, top left of data developer, then 
those items that where just assigned will disappear from your grid.  To see those items just assigned, you can set 
the user combo box to that user.


 


**Assign items to yourself:**


 


·         
After you have selected the items you want to assign to yourself from the grid, the user can right click and select **Assign Item to “User-Name”.** This will automatically assign items to 
you.


 




**Un-assign Items:**


 


·         
After you have selected the items you want to un-assign from the 
grid, the user can right click and select **Unassign** **Item.** This 
will automatically un-assign items.
