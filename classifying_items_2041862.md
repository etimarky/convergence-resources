**Viewing 
Attachment URL’s**


 


The 
Data Developer allows users to see and launch any url’s an item maybe 
associated too.  Some items may have URL 
attributes while other items maybe associated to another item, via a 
collection, and the members of the collection may have URL attributes.  Example: Link Part to Document to Attachment, 
where the attachment items have the URL’s.  
  
  
 
The Data Developer can show all of these URL relationships from the main 
grid.  A user doesn’t have to have an 
item assigned to them to view an items URL’s.  
The Data Developer will show information on each item in a collection, 
including item number, description and qualifier.  
 
 


 


**Viewing Attachment URL’s:**


 


- Retrieve the item or items you want to view from 
 Data Developer.
- Select the first column of the item and right 
 click.
- At the bottom of the right click menu will 
 display the available URL’s to launch for that item or for any related 
 items via collections that may have URL’s associated to them.
- Mouse over each item to see the items description, 
 qualifier and item number.
- Select the URL that you may want to launch based 
 on the active links displayed
