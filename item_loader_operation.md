**Item Loader Operation**


 


The Item Loader module operates in three modes. To review these modes, select these topics:


 


- To create a batch of new items and load Item Master data into the classification, review the procedure for [New Batch Mode](hs3041.md)

 


- To add new items to an existing batch, review the procedure for [Regenerate Mode](hs3043.md).

 


- To add technical attribute values, review the procedure for [Import Mode](hs3042.md).
