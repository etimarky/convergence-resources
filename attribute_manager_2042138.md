**Attribute Manager**


 


An attribute is a descriptor for an item or a category. A 
value is assigned to an attribute for an individual item. This value and the value 
of other attributes assigned to the item contribute to the item's uniqueness. 
Attributes are first added to the Master Attribute List in the Attribute 
Manager where the attribute's characteristics are defined. Once its 
characteristics are defined, and the attribute is approved, it can be assigned 
to a category in the Classification Manager module.
