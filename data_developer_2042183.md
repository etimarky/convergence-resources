**Data Developer**


 


The Item Loader Module is used to load basic information and 
attribute values for a batch of data.  The data then needs to be checked 
and in some cases, cleansed, modified or enriched before it is released. The 
Data Developer Module allows the assigned user to check, modify or verify the 
attribute values for each data item assigned to him.  The Data Developer 
is also used to view a batch of data.
