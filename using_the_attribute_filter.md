**Using the Attribute Filter**


 


The Attribute Filter provides a mechanism for selecting a subset of attributes based on specific criteria. Criteria includes any combination of Attribute Name, Attribute Description, Data Type, Unit Type, Group and/or Status.


 


To search the Master Attribute list by Attribute Name, type the search string in the **Words in Name** field. To search the Master Attribute list by Attribute Description, type the search string in the **Words in Description** field.


 


A Search String may be entered using the following rules:


 


- The attribute list may be searched by any word used in the Name or Description field
- Partial words may be entered by using an percent sign (%) to denote missing letters; For example, dia% will return attributes beginning with the word "diameter". %dia% will return attributes with "dia" embedded in the name or description
- The Attribute Filter is not case sensitive
- Multiple words must be separated by a space

 


To search by **Data Type** or **Unit Type**, press the down arrow to the right of the Data Type or Unit Type field and select the desired type.


 


To search by **Group** or **Status**, press the down arrow to the right of the Group or Status field and select the desired value.


 


Press **apply filter** (or **Filter**) to begin the search. 


 


The filter operation can be repeated using additional criteria to further narrow the search results.


 


To restore the full Master Attribute List, press the **clear filter** (or **Show All**) button.
