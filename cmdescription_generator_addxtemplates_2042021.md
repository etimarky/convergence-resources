**Description Generator Add Multiple Templates**


 


**Description Generator** now allows users to add multiple 
templates for each category.  


 


**Add multiple templates:**


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_AddXtemplates_files/image001.jpg)




 


1.     
To add a template, click the “Add Template” icon ![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_AddXtemplates_files/image002.jpg)




2.     
Enter the name of the template in the Template Name box ![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_AddXtemplates_files/image003.jpg)




**Copy 
Templates:**


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_AddXtemplates_files/image004.jpg)




1.     
To copy existing templates, select the desired templates and click 
on the “Copy” icon ![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_AddXtemplates_files/image005.jpg)




2.     
Click on the “Paste” icon ![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_AddXtemplates_files/image006.jpg)




3.     
If desired change the name of each template in the Template Name 
box ![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_AddXtemplates_files/image007.jpg)




4.     
Edit the existing template from there by clicking in the lower box 
and making the desired changes


5.     
Save the template by clicking on the ![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_AddXtemplates_files/image008.jpg)




 


**Delete 
Templates:**


1.     
To Delete a template that is no longer needed, simply select the 
desired template and click the “Delete” icon ![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_AddXtemplates_files/image009.jpg)




 


Notes: There is no limit on the number of 
templates per category.  Templates cannot be edited or deleted if the category 
has been approved.
