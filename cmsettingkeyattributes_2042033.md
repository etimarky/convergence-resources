**Setting Key Attributes**


 


Key attributes allow you to define the 
important/critical attributes for items that are in a category. Different parts 
of the system will use key attributes to provide features, such as SmartFind 
showing key attributes as filters for items in a particular category.  There 
are two methods of defining key attributes, in the category attribute edit 
dialog or by selecting the attributes in the classification manager and using 
the right click menu.


**In order modify key attributes you 
must have the category developer role assigned to your user.**


**Setting An Individual Category 
Attribute As Key**


In the classification manager, if you 
right-click on an attribute and choose edit or if you double click on an 
attribute you will receive the category attribute edit dialog.


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-SettingKeyAttributes_files/image001.png)




Notice under the settings tab there is a 
checkbox that allows you to specify whether or not the attribute is a key 
attribute.  This setting like all others in this dialog will propagate down the 
tree if you are setting this on a non-terminal node.


**Settings or Removing Key Attributes 
At Category Level**


When selecting a category, the attributes in 
that category are shown to the right.  If you select an attribute or 
multi-select several attributes you can then right-click and a menu will show 
up allowing you to set attributes as key attributes.


**![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-SettingKeyAttributes_files/image002.jpg)**


In order to set selected attributes as key 
attributes at least one of the attributes selected must not be a key attribute 
already.  In order to remove attributes as key attributes at least one of the 
attributes selected must be a key attribute already.


**Viewing Key Attributes**


When viewing the attributes assigned to a 
category in the classification manager, you will see key attributes denoted by 
a blue key.


**![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-SettingKeyAttributes_files/image003.jpg)**
