# Sort Values

Sort\_Values - Design For Retrieval (DFR) Help

## Sort Values

&#x20;

Go to the Allowed Values page.

Values can be sorted by clicking on Sorting Mode

![](.gitbook/assets/image\(843\).png)

&#x20;

Values can be sorted alphabetically by Clicking on Alpha-Sort

&#x20;

![](.gitbook/assets/image\(844\).png)

&#x20;

Or values can be moved around by holding the left click of the mouse to drag the values around.

Also, all the highlighted icons on the right can be used for sorting.

If you click on the trash icon, a value can be deleted.

![](.gitbook/assets/image\(845\).png)

&#x20;

Click on Display Mode to make all changes.
