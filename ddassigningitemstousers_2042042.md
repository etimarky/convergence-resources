**Assigning 
Items to Users**


 


 


Items can be assigned to the current user or to other users listed 
in the DFR catalog from the Data Developer. 


 


The user who makes the assignment must have the **Item Assign** 
role to perform this function. This role is set in the User Manager module by 
the Admin or someone with the **Catalog Administrator** Role.


 


To assign an item (or items) in Data Developer, select the item(s) 
using the shift or Ctrl key, right click, and choose Assign Item.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Assigning-Items-to-Users_files/image001.jpg)




**Example**: Assigning 3 items to a user.


 


Once the Assign Item has been clicked, the use will have the 
option to choose the Assignee:


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Assigning-Items-to-Users_files/image002.jpg)




**Example**: Choose the user you would like, and click the OK button.


 


If 
the user is just assigning the item to him/her, his/her username will appear 
directly under the Assign Item command as Assign Item to *username*.
