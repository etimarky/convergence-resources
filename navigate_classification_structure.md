# Navigate Classification Structure


There are multiple ways to navigate the tree.


 


In classification, click on the orange-colored icon; this will be located on the upper left side of the screen.  

 


![](assets/image(1169).png)




 


Root will be the first selection once the window expands.


Select the arrows next to the folder to expand the desired category OR


The search bar at the top can be used to filter quickly as well.


To pin the expanded window, select the thumbtack icon.  

 


![](assets/image(1170).png)




 


**OR**


 


In classification, select the black arrow next to the Root folder on the upper left side of the screen.


Select the desired category from the expanded menu screen.  

 


![](assets/image(1171).png)




 


Navigate the breadcrumb to the desired category and click on it.  

 


![](assets/image(1172).png)
