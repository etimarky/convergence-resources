**Description**


 


Keep the definition fairly generic.   For example, "The maximum pressure rating" would be an appropriate description of Pressure Rating, and would allow the attribute to be appropriately used on other products in the classification tree. 


 


"The maximum pressure rating on a hydraulic pump" is too specific; If someone wanted to use Pressure Rating for a product other than a pump, the description implies that it would not fit.


 


In the following example, a specific description is appropriate because pump is used in the attribute name: Pump Material - The type of material that is used to make the pump.
