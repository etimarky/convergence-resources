# Search by Classification


Searching for parts can be done by Classification.


 


To search by Classification/Category, select Classification from the drop-down menu.


![](assets/image(330)_1.png)




 


Enter the classification name. Select a category from the drop-down list that appears or select Search.


![](assets/image(331)_1.png)




 


The Classification search will allow you to find all categories that contain the word "valves".


On the left side of the page, you can see the categories that contain that word; there may be more sub-categories under those.


Clicking on any of the item numbers will take you the main part where there's a category with the word used for the search.


![](assets/image(563).png)
