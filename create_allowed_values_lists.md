## Create Allowed Values Lists (AVL)

&#x20;

&#x20;

Go to the Allowed Values page.

In "Add New AVL...", type desired name for the Allowed Value List.&#x20;

&#x20;

![](.gitbook/assets/image\(885\).png)

&#x20;

Once you type the desired name, click on Apply or press Enter on your keyboard.

&#x20;

![](.gitbook/assets/image\(569\).png)

&#x20;

This is how the AVL is displayed in Convergence PIM.&#x20;

&#x20;

To add values to the Allowed Values List, select " Edit Values"

For more on adding values, see [How to Edit an AVL.](create_allowed_values_lists.md)
