# Create Allowed Values Lists (AVL)


 




<iframe src="https://www.youtube.com/embed/Oj0RKErCIwI/?wmode=transparent" width="560" height="315" frameborder="0" allowfullscreen></iframe>


 


Go to the Allowed Values page.


In "Add New AVL...", type desired name for the Allowed Value List. 


 


![](assets/image(885).png)




 


Once you type the desired name, click on Apply or press Enter on your keyboard.


 


![](assets/image(569).png)




 


This is how the AVL is displayed in Convergence PIM. 


 


To add values to the Allowed Values List, select " Edit Values"


For more on adding values, see [How to Edit an AVL.](#)
