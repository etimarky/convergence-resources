**Customizing 
the Data Display**


 


The Data Developer Module allows the user to hide columns and sort 
data to expand the work area and increase the efficiency of data manipulation. 
Use the **View** drop-down menu on the tool bar to manipulate the contents 
of the screen.


 


**To hide the Toolbox column:**


 


·            
Move the cursor to the X in the upper right corner of the **Toolbox** 
column on the left side of the screen


·            
Press the **X** button.  Note that 
the Toolbox column on the left side of the screen has been removed.


 


 


**To hide the Category Tree column:**


 


·            
Press the **View** button on the tool bar at the upper left 
corner of the screen


·            
Select the **Hide category tree** button.  Note that the category tree column has also been removed.


 


The screen is now formatted to show the maximum screen area in 
spreadsheet format. However, there are several columns displayed with no data 
to work on and others with data that are not in view.  Columns 
not needed for editing can be hidden from view.


 


**To maximize the data display:**


 


·         
Press the **View** button


·         
Select the **Show/Hide Columns** option


·         
Using the **Show/Hide Attributes** window, attribute columns 
may be selected to be displayed or hidden


·         
To hide an attribute column, select the attribute or attribute (s) 
and then select Hide Y/N. Attributes that are hidden will show up in italic 
font. To unhide attributes, select the attribute or attributes (s) and then 
select Hide Y/N


 


 


The order of the displayed columns can also be changed, moving 
target attributes into a preferred order for editing.


 


**To change the order in which data columns are displayed:**


 


Click on the attribute name, holding down on the key on mouse, 
slide the attribute column to the right or left. This will move the attribute 
column to your desired position.  The data is now ready to be edited.


**To reset the columns to the original order:**


 Press the **Reset Columns** button at the top of the 
screen


 


**To Format Row Height and Column Width: (gang selection)**


 


·         
Select all of the row(s) you want to adjust the row height.  Drag 
the last row down to the desired row height.  You will notice the other rows 
will adjust the same way.


·         
Select all of the column(s) you want to adjust to column width.  
Drag the last column to the desired column width.  You will notice the other 
columns will adjust the same way.


·         
Both row and column formatting can be saved with a View.  
Formatting can be removed only once reset columns is set, clear all filters 
button does not clear any formatting.
