**Commit 
or Cancel data in the Detail Grid**



If you have added, deleted or modified data in the detail 
grid, you have the option to commit those changes, or cancel them.  To commit 
changes, click the “Commit” button.


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Commit-or-Cancel-data-in-the-Detail-Grid_files/image001.jpg)

Example: Committing 
changes to the detail grid.


 


To cancel the changes you made, click “Cancel”. 


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/DD-Commit-or-Cancel-data-in-the-Detail-Grid_files/image002.jpg)




Example: Cancelling changes made to the detail grid.
