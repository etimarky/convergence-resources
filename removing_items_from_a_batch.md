## Removing Items from a Batch

There are two main ways to remove Items from a Working Batch in Convergence PIM:

&#x20;

The first way to remove items from a batch is to: Remove Directly from Data Developer

&#x20;

First, Open Convergence PIM Thick Client and open the Data Developer Module.

Click the "All Batches" Buttons to select a batch that you would like to remove parts from.

&#x20;

![](.gitbook/assets/image\(1246\).png)

&#x20;

In the Batch Pop Up Window, select the batch you want to open.&#x20;

Press the "Select Batch/Deliverable" button to open that Batch in Data Developer.

&#x20;

![](.gitbook/assets/image\(1247\).png)

&#x20;

Locate and Select the item you want to remove from the batch.&#x20;

Right click on the line of data for that item.

From the Dropdown, select "Remove Item from Batch or Deliverable." As long as the Item is in at least 1 batch additional batch, it can be removed from the current one.

&#x20;

![](.gitbook/assets/image\(1248\).png)

&#x20;

&#x20;

The second way to remove items from a batch is to Use the Manage Batches Window

Begin by following the same steps as above except, instead of selecting to remove the item from the batch, select the "Manage Working Batches" option from the menu.

Once you are in the Manage Existing Batches Screen, select the Batch  you want to edit.

Below, you can select/multi select any parts you want to remove, and click "Remove Item(s)"

&#x20;
