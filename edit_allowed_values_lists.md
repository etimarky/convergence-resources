# Edit Allowed Values List(AVL)


 


Editing an AVL is split into two sections: editing the name of the list itself, and adding/editing values. 


 


In the Classification properties under the chosen categories, click on Details.


All values stored in AVL are displayed under Values.


In Where Used, you can see under which categories the AVL is located.


 


![](assets/image(832).png)




 


Click on Close to go back to the Classification page and select Edit to begin editing your AVL.


 


Click on the drop-down next to Details and select Edit.


Also, an AVL can be unassigned by clicking on Unassign.
