**Description Generator Assigned Items Only**


 


The category processing action **Description Generator**, now 
defaults to only allow updates to items owned by the user.  This stays with the 
philosophy of Data Developer that an Item must be assigned to a user in order 
for that item to be updated by that user.  There is a setting that will 
override this setting.  However only those with the appropriate permissions to 
update Options Manager can update this setting.


 


**Restrict to my items only:**


 


Once the user is happy with their template and it is saved, select the 
Run radio button at the top left of the screen. 


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_RunFilesAssigned_files/image001.jpg)




 


You will see the Restrict to my items only check box is greyed 
out.  This is checked by default.  The general user will only be able to update 
their own items assigned to them.  In order to override this setting, 
AllowGlobalDescriptionChanges can be set to true in Options Manager.  This will 
allow all items in a batch to be updated by any user.  


 


![](/userfiles/13115/16830/ckfinder/images/import-d2f34nel28gg/CM-Description_Generator_RunFilesAssigned_files/image002.jpg)
