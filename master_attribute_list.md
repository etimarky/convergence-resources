**Master Attribute List**


 


The Master Attribute List is made up of all attributes created in the Attribute Manager tool. This list is diplayed in alphabetical order in the Attribute Manager tool. Use the scroll bar on the right to move up and down through the list of attributes to find specific attributes. 


 


Use the [Attribute Filter](hs2400.md) to narrow the list or to search for a specific attribute.


 


The list can be accessed from the Category Manager tool by selecting a category, right clicking and selecting Edit Category. The Attribute Mapping window is displayed. The Attribute Filter can also be used in this window.
