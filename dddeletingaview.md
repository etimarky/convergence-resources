**Deleting 
a View**




To 
delete a view, click the “Views…”  button.




**![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Deleting-a-View_files/image001.jpg)**


**Example**: Clicking the views 
button.


 


After 
clicking the views button, a dialog is presented for the user to select a 
previously saved view.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-Deleting-a-View_files/image002.jpg)




**Example**: To delete a view, 
select it from the list of available views, and click the “Delete” button.


 


**Note**: Only a Catalog Admin 
can delete public views.
