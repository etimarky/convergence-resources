# Create Parts

Create\_Parts - Design For Retrieval (DFR) Help

## Create Parts

SmartCreate is used to create new individual parts.

&#x20;

Select SmartCreate from the menu on the top left.

&#x20;

![](.gitbook/assets/image\(1120\).png)

&#x20;

Select a category from the category data model located on the left side of the screen. Attributes for the selected category will be displayed on the right.&#x20;

Locate the desired category and select "Continue."

&#x20;

![](.gitbook/assets/image_30.png)

&#x20;

The third screen will display only Key Attributes for data entry.&#x20;

As each value is entered, the possible duplicates will appear at the bottom.

&#x20;

If the part already exists, it can be duplicated by utilizing a **qualifier** to essentially tell DFR that it is a new part.&#x20;

![](.gitbook/assets/image_31.png)

&#x20;

Once there are no possible matches, the item will be created as New.&#x20;

Select Continue.

&#x20;

![](.gitbook/assets/image_32.png)

&#x20;

Begin by entering Item Number and Item Description.&#x20;

Minimally, the Item Number needs to be entered in order to save the new part.  Then use the attribute table and fill in any applicable attributes.&#x20;

Users may utilize the ability to add to a batch, deliverable or item access group from this screen as well.

Select Save when complete.

&#x20;

![](.gitbook/assets/image_33.png)

&#x20;

The next screen will show the part has been created.

&#x20;

![](.gitbook/assets/image_34.png)

&#x20;

To verify the part created exists in DFR, select SmartFind and search for the part.

&#x20;

![](.gitbook/assets/image_35.png)
