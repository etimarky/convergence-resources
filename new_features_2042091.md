## Summary of New Features:


3.6.999 was upgraded with delivery of bug 
fixes noted in this release. There were 2 enhancements.  


·         
WebDFR 
is to honor the Read-Only attribute flag.  


·         
Allowed 
Values Manager includes an import function for Local and Global Lists. 


·         
DFR 
Client installer now allows installation per user which automatically goes to 
the user’s application folder.


·         
DFR 
Client installer now creates desktop and start menu shortcuts specifying the 
Target.
