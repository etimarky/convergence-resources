**Mapping 
Creation Process**



When a user opens a module road map they will see both 
modules (listed in the middle of the road map) and module variants (starting in 
the first column).  Each module can have multiple module variants 
associated with them.  A line across the road map will separate data 
between modules.  Users have the ability to create road maps with the 
module variants list in the first columns.  Module variants that are newly 
created will be listed at the bottom of the first column (see example below).

## How to create a road map


A user can create additional replacements in future years 
for each module variant either by going across the page or mapping to other 
rows.   Here are the different options for creating a road map for a 
variant.  All options are available by right clicking. 


 



**Creating a replacement (same row) –** You can right 
click and select **Copy into next year** or use the control keys**.** Note 
you can only create one replacement at a time or move one space to the right.



**Creating a replacement (different row)** –First 
select the variant year. Then drag your cursor to the row containing the future 
variant year, keeping your cursor down until the replacement variant is 
selected.  



**Moving vertically –** Selecting the variant year and 
dragging your cursor up or down allows a user to move a row up or down 
vertically.  Make sure to keep your cursor down until you have reached the 
destination of your new row.  You may have to create a new blank row 
before moving other existing rows vertically.



**Creating new rows –** To create a new row, use the 
right click menu and select Insert Row.



**Deleting Replacements** – There is an option in the 
right click menu to delete replacements called Delete item-yr.  If you 
just want to delete the last variant in your row as an example, you can select 
it and select Delete item-yr.


 


**No Replacement** – If a variant has a replacement to 
the right, the user has the option to break that link by selecting No 
Replacement.


 


**Cancel Edits** – If you need to undo your last edits on 
a variant year, select Cancel edits.


 


**Details** – To view the details of a variant select 
Details. An alternate way is to select the variant, and press *F2* on the 
keyboard.


 


**Copy into Previous Year** – If you don’t have a variant 
to the left, you can use this feature from the right click menu.
