**Edit 
System Attributes**



Data 
Developer allows a user to edit system attributes.  This feature is available 
to those with the Category Administrator role.  The item must be assigned to the 
user, not be Pending or Approved and only a single item must be selected.  


 


In 
order to edit the attributes, select the item and right click the item.  Select 
Edit System Attributes.    


 


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-EditSystemAttr_files/image001.jpg)





This 
will display the Edit System Attributes dialog box.  Here the user can edit the 
system attributes including Item Number, Item Description, Legacy Item Number, 
Revision, Qualifier and Release Date.  Click the ![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-EditSystemAttr_files/image002.jpg)

 button.   


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/DD-EditSystemAttr_files/image003.jpg)
