**Removing a Working Batch**


 


Removing a working batch removes the association between an item and a batch.  Removing a working batch does not delete items or data.


 


**To remove working a batch:**


 


- In the Item Browsers select Manage Working Batches...

 


- In the Managing Working Batches window select the working batch you wish to delete

 


- Select the Delete Batch button
