



Data\_Type - Design For Retrieval (DFR) Help



**Data Type**


 


Attribute Data Type describes the kind of data stored in the attribute. The data can be a [String](hs2210.md), [Numeric](hs2220.md), [Integer](hs2230.md), [Fraction](hs2270.md), [Date](hs2250.md), [Boolean](hs2240.md), or [URL](hs2260.md) or custom value.


 


In addition, the Data Type of Items is used to indicate that the attribute is a collection attribute.  Items collections are described in the **Item Collections** topic.


 


The default data type is String.
