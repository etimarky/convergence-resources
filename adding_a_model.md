# Adding a Product


A product in SmartTeardown refers to an item, appliance, part etc. such as a "dryer, washing machine, dishwasher, etc." 


 


Navigate to SmartTeardown. 


The first screen that appears will be where the user adds a product/model. 


 


![](assets/image(67).png)




 


Fill in all pertinent information. An image can be added at this step as well. 


All information must be filled in. 


If features were identified in Family Management, these will appear as an option on this page too. 


 


![](assets/image(68).png)




 


![](assets/image(69).png)




 


Once all information is filled out, the model will be added to the designated family, which is set up beforehand in Family Management. 


In this case we are adding a dryer model, so we will see Dryers show up on this front page now. 


 


![](assets/image(70).png)




 


Drilling into the Dryer family, we will see all manufacturers for dryers.  In this example, GE Appliances is the only manufacturer available but if we go a step further, we will see all models that share that same manufacturer. Any feature sets that were identified in Family Management will also appear so that users can easily filter on specific features which will generate results of just those models and manufacturers. 


 


![](assets/image(71).png)




 


Selecting a model will show all model information previsouly set up such as name, teardown year, an image, etc. 


Selecting the orange operations button will allow the user to either Edit, Export, Delete or add Attachments. 


Below the general model information, Hierarchy can be edited or built out. Once again, there cannot be more than 5 hierarchy levels. 


 


![](assets/image(72).png)




 


Heiarchy will appear gray when there is no fastener or segment data attached to the model. 


When blue, this indicates that there is information on that level. 


 


If a gray category is selected, there will be a message indicating that there is no segment data.
