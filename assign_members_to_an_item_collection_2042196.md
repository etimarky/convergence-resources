**Assign Members to an Item Collection**


 


Assigning members to an item collection is done with the Item Browser. Before members can be assigned to an item collection, an [item collection attribute](javascript:popuplink('hs8100.htm')) must be assigned to a category where the item is located.


 


**The item must be assigned to you before you can add items to a collection membership.**


 


**To assign members to an item collection:**


 


- Find the item for the collection using the standard search functions.

 


- Once you have located the item, select the **Collections** tab.

 


- Select the item collection from the **Collections to search** pull-down menu.

 


- Select the **Search** button.

 


From the **Items** pull-downmenu, select **Detail Items**.


 


- Select **Add Items...**.  This will bring up another window very similar to the item browser window.

 


- Use the **Add Items** browser window to search for the item(s) that you want to add to the item collection.

 


- You can select multiple items using shift-click or cntr-click.

 


- Once you have the items selected, select **OK** and the Add Items window will close.

 


- Select the **Commit** button in the Detail Items window to assign the members to the collection.

 


**Hint:** To quickly select and assign multiple items, create a working batch of the items and then select the working batch in the Add Items window.
