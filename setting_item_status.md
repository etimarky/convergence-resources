**Setting Item 
Status**


 


The 
Data Developer allows certain users to change status of an item.  Only those users that are either catalog 
administrators or item approvers can change the status of items.  An item doesn’t have to be assigned to a user 
to change its status. 


  
 
 


 


**To set Item Status:**


 


- Retrieve the item or items you want to change 
 status to from a category in Data Developer.
- Select the first column and select the number of 
 rows of items you want to change status for.  A user can select items individually 
 using the control key or they can select a group of items using the shift 
 key to highlight multiple rows.
- Right click on the first column to bring up the 
 item status menu.  Select one of the 
 status’s that are not grayed out.  
 Check for mixed status, this may make it more difficult to change status 
 on a group of items.
- There are three possible statuses a user can set 
 including: in-work, pending, and approved.  
 Once an item is set to either pending or approved, it can no longer 
 be edited.
- A user can set the status back from approved to 
 pending or back to in-work if they choose.
- **You can only 
 change status on items that are visible in the Data Developer**.  If you need to change status on a large 
 group of items, use the Item Browser.
