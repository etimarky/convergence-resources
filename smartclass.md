# SmartClass


**SmartClass** is the Convergence PIM module where all updates related to the data model are performed, including changes to:


- Category structure/schema
- Attributes
- Attribute groups
- Allowed Values Lists (AVLs)


 


Click on the SmartClass icon to access the module.


 


![](assets/image_51.png)




 


**SmartClass Functions**


On the left-hand side of the page, there are 4 icons that allow you to change what part of SmartClass you are working in.


 


![](assets/image_52.png)

 The folders icon is for **Classification/Schema** and is where you can view the category tree.


![](assets/image_53.png)

 The single tag icon is the **Attribute**library, where you manage all attributes for this catalog.


![](assets/image_54.png)

 The multiple tags icon is the **Attribute Group** library, where attribute groups can be created and attributes can be assigned to groups.


![](assets/image_55.png)

 The bulleted list icon is the **Allowed Values List**library, where you can manage and assign legal values for attributes.
