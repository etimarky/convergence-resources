



Attachments - Design For Retrieval (DFR) Help



**Attachments**


 


Attachments provide a way to associate documents with a category or node. All the items associated with the node will have a link to the documents that are listed as attachments.


 


In order to use attachments, the Classification must have a category with the path **Root\DFR Administration\Attachments**.


 


The Attachments category must have an attribute call "URL" set up as a string attribute type. The URL for the attachment must be mapped to this attribute. 


 


**To assign or remove attachments, the category must not have an Approved status. The user must have the Category Developer or Category Approver role for the category where the attachments will be assigned.** The attachments do not have to be assigned to the user making the assignment.


 


There are two methods for assigning attachments to a category. Using the Item Browser to select attachments and add them to a category is the prefered method to avoid errors while manually typing the URL address.


 


**Using the Item Browser:**


 


- Move the cursor through the classification to expose the category where the attachments will be assigned.

 


- Open the Root\Non-Parts\Attachment category.

 


- Select the attachment or group of attachments, using the shift or control keys to select multiple items.

 


- Drag the item(s) to the target category. Notice the plus sign near the cursor as the cursor approaches the classification structure.

 


Note: The attachments must have Attachment as a qualifier.


 


**Using the Classification Manager:**


 


- Select the category where the attachments will be assigned

 


- Right click on the category name and select **Update Category**

 


- The Category Maintenance window is displayed

 


- Select the **Attachments** tab

 


- Manually type in the URL on the next available line

 


- Select the **Update** button to save the changes

 


 


To view attachments related to a category. Open the Attachments tab for a category; click on the attachment number, not the URL, to view the attachment.


 


All the items associated with node will have a link to the documents that are listed as attachments.


 


To delete an attachment to a category:


 


Deleting an attachment from a category will delete the attachment from all subordinate categories.
