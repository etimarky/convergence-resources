# Copy from Another Part

Copy\_from\_Another\_Part - Design For Retrieval (DFR) Help

## Copy from Another Part

In SmartFind, you are able to copy all attribute values from one item and paste them over another item.

This feature will override all existing attribute values to a part with the values "copied" from the desired part.

&#x20;

Go to any category at any level or simply select a part.

In this case, Scroll Compressors is shown and the part that will be updated is "001-27421" and we will copy the attribute values from "001-27422".

**All attribute values from "001-27422" will be copied, except for the values that are blank.** Those null values will not override the values in "001-27421", instead the existing values in "001-27421" will be kept.

Click on the part "001-27421".

![](.gitbook/assets/image\(1108\).png)

&#x20;

Review all attribute values and click on Edit.

![](.gitbook/assets/image\(1111\).png)

&#x20;

Click on Copy Anther Item.

![](.gitbook/assets/image\(1110\).png)

&#x20;

Enter the item number where the data will be copied from; item number "001-27422" is typed.

Click on Search.

![](.gitbook/assets/image\(1112\).png)

&#x20;

Make sure Classification Data and Attribute Data are selected.

Click on Copy.

![](.gitbook/assets/image\(1116\).png)

&#x20;

Now, all attribute values are shown with all the new updates as shown.

Click on Save to make all changes.

![](.gitbook/assets/image\(1117\).png)
