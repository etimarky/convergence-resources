# Add Category Images


This feature allows users to display Images under Categories to help them identify their products/parts.


 


In classification, locate the desired category.


![](assets/image(1069).png)




## 


Now all that the categories are being displayed, select the category where the image will be uploaded.


In this case, the Compressor Accessories category is selected.


Hover over the icon on the top right of the page and click on the Edit icon.


![](assets/image(1072).png)




 


Now the category properties can be updated.


Hover over the category image and select the Upload icon to add a new image.


Also, an existing image can be removed by clicking on the Remove icon to delete the image.


![](assets/image(1071).png)




Look up the folder where your image is stored


Select the image and click on Open


Note: JPG and PNG files are supported
![](assets/image(556).png)




 


Click on the Save icon to make changes or click on the Cancel icon to undone changes.


![](assets/image(1074).png)
