## Edit Attribute Details

SmartClass gives the user the ability to edit specific category attributes details as they see fit. This only applies to the current category and will not edit the master attribute. This can be done at a leaf category or parent category level. If done at a parent category level, the changes will cascade down to the leaf categories if "cascade" is turned on.

&#x20;

In classification, locate the desired category.

Once the category attributes appear, select the attribute to be edited.

![](.gitbook/assets/image\(713\).png)

&#x20;

Select the arrow on the right side or double click on the desired attribute to open up attribute details.

Select Edit to begin editing attribute information.

![](.gitbook/assets/image\(714\).png)

&#x20; &#x20;

By typing under the attribute name, the Description can be updated.

By clicking on the checkboxes, you can make this attribute a Required, Read-Only, Allow DNA, or Key attribute under this category.

Also, the attribute status can be updated under that category.

Click on Save to make changes.

&#x20;

![](.gitbook/assets/image\(715\).png)

&#x20;

Properties can be updated by clicking on the arrow next to Properties.

* These Properties are for Advanced Users.

Attribute Groups can be updated by clicking on +Add or -Remove.

Click on Save to make changes.

![](.gitbook/assets/image\(613\).png)
