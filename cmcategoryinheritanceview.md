**Viewing Category Inheritance**



The 
category inheritance view shows information about inhertance options of each 
category in a graphical manner.


 


**To 
view category inheritance:** 


 


·         
While 
Classification Manager is open, select a category which inherantance 
information is to be viewed for.


·         
Select 
the **View** menu followed by **Category Inheritance**.  This will 
display a dialog with a tree representing the category structure.  


·         
Customize 
the view by selecting the coorespoing buttons in the ribbon bar to show the 
information desired.

**To change inheritance options:**


·         
Right click on the category to display which options can be 
changed.  


·         
Select which option to change.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/CM-Category-Inheritance-View_files/image003.png)




 


NOTE: Options are dependent on what is chosen to be viewed via the 
buttons in the ribbon bar.  Only users with the catalog role can change 
inheritance options.


 


 


**To export category inheritance:**


·         
Launch the category inheritance dialog.


·         
Select which inheritance options to view.


·         
Select **Export** then **Excel**. 


·         
Choose a location and file name for the exported file.


·         
Select **OK**.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/CM-Category-Inheritance-View_files/image004.png)
