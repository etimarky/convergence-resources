# Load Values from a Text File

Load\_Values\_from\_a\_Text\_File - Design For Retrieval (DFR) Help

## Load Values from a Text File

Values can be imported from .csv or .txt file by clicking on Load Values.

&#x20;

Go to the Allowed Values page.

Select the desired file and click on Open.

![](.gitbook/assets/image\(846\).png)

&#x20;

Then, click on Load Values to add all values from the file.

![](.gitbook/assets/image\(585\).png)

&#x20;

Now all new values have been loaded in the AVL.

![](.gitbook/assets/image\(840\).png)
