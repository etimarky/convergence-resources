# Edit Categories


Editing categories is a quick and simple process done in SmartClass through the Classification module. These changes can encompass category images, names, descriptions, and properties.


 


In classification, locate the desired category to be updated.


![](assets/image(961).png)

  

 


Once the category is selected, click the orange icon in the top right of the category- this is the Edit icon.


![](assets/image(960).png)




 


The edit icon will open up options to edit features:


- [Edit Category Properties](#)
- [Edit Category Details](#)
