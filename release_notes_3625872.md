# Release Notes




---


 


v15.3 (May 2025)


 


*Enhancements*


**SmartClass**


- Users may now be able to convert item categories to a tiered category either one at a time or an entire subtree.
- Users may now set the edit tier of a tiered category attributes.
- When viewing category attributes, the user may now filter on inherited attributes.


**Importing/Exporting**


- Users can now import tier information of category attributes.
- Users can now import category type information.
- Performance enhancements on the imports were made to reduce timeout issues.
- Category exports now include tier information.
