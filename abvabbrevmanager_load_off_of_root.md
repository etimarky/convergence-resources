**Abbreviation 
Manager Dictionary**


 


 


The 
*Abbreviation Manager* allows users to manage their Abbreviation Manager 
Dictionary in a custom drop-in.  This feature is very simple to use.


 


This 
tool can be found on the main toolbar by Tools > Abbreviation Dictionary.


 


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/ABV-AbbrevManager_(load_off_of_root)_files/image001.jpg)




The 
following modules must not be open while working in the Abbreviation Dictionary 
tool.  


·         
Classification 
Manager


·         
Attribute 
Manager


·         
Data 
Developer


The 
tool will display a simple message to close out of these modules if open.  


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/ABV-AbbrevManager_(load_off_of_root)_files/image002.jpg)

  
  
Once the tool is open, the user has the option to filter if the list is 
too long.  The filter will automatically bring the user to the list by the 
first characters typed.  So if the user types in SER, the list will place the 
cursor to the first time that “BEGINS” with SER.  It is a very simple filtering 
option.  


To 
add an abbreviation, click on a cell and type the value into the Value Column 
and enter a corresponding abbreviation into the Abbreviation column.  It will 
be stored here until the user calls it.  When the user is in Allowed Values 
Manager, the user can enter in that particular value as an Allowed Value for an 
attribute.   When the value is entered, the Abbreviation automatically inserts 
into the Abbreviation column in Allowed Values Manager as the corresponding 
Abbreviation for that Value.  


To 
Edit an abbreviation, just click in the Abbreviation column and change the 
value.  It will automatically update in the Allowed Values Manager.  


To 
Delete a Value in the Abbreviation Dictionary, select the value, making sure 
the entire cell is selected.  Hit the delete key.


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/ABV-AbbrevManager_(load_off_of_root)_files/image003.jpg)




The 
user will see a confirmation message to click Yes to confirm the delete.  This 
will delete the abbreviation value from the Allowed Values lists it is 
associated.  The Allowed Value will still remain.


![](/userfiles/13115/16830/ckfinder/images/import-cbg4qpn20hv5/ABV-AbbrevManager_(load_off_of_root)_files/image004.jpg)
