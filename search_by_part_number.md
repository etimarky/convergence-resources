# Search by Part Number

Search\_by\_Part\_Number - Design For Retrieval (DFR) Help

## Search by Part Number

Searching for parts can be done by Part Number.

&#x20;

To search for a part by Part Number, select Part Search from the drop-down menu.

![](.gitbook/assets/image\(327\).png)

&#x20;

Enter the part number and choose the part number from the drop-down list that appears or just simply click on Search.

![](.gitbook/assets/image\(329\).png)

&#x20;

In this case, if you select the part number "oo1-00001" the Part Search option will take you to the main part and display all its details.![](.gitbook/assets/image\(561\).png)

&#x20;

In this case, if you type "001" and click on Search, all parts numbers that begin with "001" will be displayed.

![](.gitbook/assets/image\(1006\).png)
