**Pruning and Grafting a Category**


 


A portion or branch of the classification tree and its subordinate structure may be moved to a new location by pruning and grafting the branch.  


 


**Because this operation can cause major changes to the classification structure Category Developer and Category Approver roles are required to prune and graft.**


 


**To prune a branch:**


 


- Highlight the parent category of the branch to be moved.

 


- Press the right mouse button and select **Prune**. Note that a tree has been displayed next to the selected category. This identifies the branch to be moved.

 


- Navigate through the classification structure to the location where the pruned category will be moved. Select the parent category and press the right mouse button.

 


- Select **Graft**. Note that the pruned category and its children have been moved to the new location.
