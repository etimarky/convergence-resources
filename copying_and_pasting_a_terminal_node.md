**Copying and Pasting a Terminal Node**


 


 


Only terminal nodes may be copied.


 


**To copy a terminal node:**


 


- Highlight the terminal node to be copied. Press the right mouse button.

 


- Select **Copy Category.**

 


- Move the cursor to the parent category for the node to be copied and select the parent category.

 


- Press the right mouse button and select **Paste Category**. The Category Maintenance window will be displayed. Notice that the parent category is displayed in the **Parent** field. The copied category has all of the attributes, properties, authorization and status of the original. Created and Last Updated fields will be updated to reflect the current time and user.

 


- Select the **Untitled Category** in the Name field. Change the Name field and any other information specific to the new node.

 


- Press **Update** to accept the changes and close the Category Maintenance window. Note that the newly copied terminal node has been added to the classification structure.

 


- Pressing **Cancel** will not remove the copied category. The new category will remain in the classification labeled as "Untitled Category" under the parent category. This category must be deleted using the [Delete category](hs1230.md) function.
