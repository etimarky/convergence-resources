**Item 
Mapping**



Item Mapping is a tool used to create and 
edit road maps for Module Types including Module and Module Variants.  The data 
for production road maps is stored in the DFR database.  Modules to variant and 
variant to component BOM relationships are read from Parts & BOMs.  What-if 
roadmaps can be created and shared but these are not saved in the DFR 
database.  These can be viewed and edited in DFR and saved to a user’s hard 
drive in XML format.  Summary reports can be viewed and printed from DFR along 
with the associated road map data.


 


ERP data for module variants and components 
will be read from other Whirlpool data sources (TBD).  Currently this data is 
manually entered and calculated.  Several features include filtering to 
graphically indicate owner group, region and status code.  Module order can be 
changed to help group modules that link to each other.  Variant data can be 
copied in to the next year.  The base year holds only read only data.
