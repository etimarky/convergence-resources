**Creating and Editing Attributes**


 



**To add an 
attribute, you need the following information:**


·         
The [**Name**](javascript:popuplink('hs2160.htm')) and [**Description**](javascript:popuplink('hs2170.htm')) for the 
new attribute 


·         
The [**Data Type**](javascript:popuplink('hs2200.htm')) of the 
attribute  


An attribute may be added, modified or deleted using the [**Add**](javascript:popuplink('hs2110.htm'))**,** [**Edit**](javascript:popuplink('hs2120.htm'))or[**Delete**](javascript:popuplink('hs2130.htm')) button at the top of the 
center column. When you select Add or Edit, the action options will change to [**Save**](javascript:popuplink('hs2140.htm')) or [**Cancel**](javascript:popuplink('hs2150.htm'))**.**


**Editing an attribute:**


Certain edits will allow the user 
to update any Description templates which will be affected by changing an 
attribute definition.  These changes include:


·         
Renaming an attribute


·         
Changing the attribute [**Data Type**](javascript:popuplink('hs2200.htm')).


·         
Deleting the attribute.


The above change types trigger 
logic which matches categories using the attribute and description templates on 
the categories which use the attribute.  If matches are found a user prompt is 
displayed allowing the user to update description templates based on the 
changes made to the attribute. NOTE: If a large amount of matches are found, 
selecting yes may result in a delay before all description templates are 
updated.
