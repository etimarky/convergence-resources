**Numeric Data Type**


 


Numeric attributes have decimal values and have a [Unit of Measure Type](javascript:popuplink('hs2221.htm')) assigned to them. Quite often, the same instance of a numeric attribute can have different units of measure; that is, an attribute called "length" can have a unit of measure of feet or meters. 


 


When selecting a unit of measure, the default **unit of measure system** is also being determined.  These systems include metric system, international system (SI), and US system.


 


Each numeric attribute will also have a [Unit of Measure](javascript:popuplink('hs2222.htm')) where, for each unit of measure type, there is a group of commonly used units. For example, an attribute called "Pressure Rating" has a unit of measure type of pressure and typical units of psi, kpsi, pa or kpa.


 


Although a single unit of measure is assigned to an attribute when it is created, the unit of measure can be reassigned on an individual data item in the Data Developer module.


 


The Attribute Manager offers commonly used units; however, if a unit is needed and is not displayed, it can be defined in the database by the database administrator and added to the pull down list.
