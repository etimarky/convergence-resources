# Deleting a Status


Deleting a status is done when a status is no longer necessary and will remove the status indefinitely. 


 


Permissions need to be set and allowed for users to utilize the catalog manager and status manager.   

These permissions include:   

-Classification Permission: Category Permission/ Edit Items  

-Catalog Permission: Access Catalog Status Module
 


Deleting a status is easily accomplished by navigating to the **[Status Mangement](#)**screen, selecting the status and then " Delete" on the status to the left. 


 


![](assets/image(940).png)




 


A status can only be deleted as long as it is not tied to any categories, attributes or parts. 
 


If a status is tied to any of the three elements listed above, an error message will be prompted:


 


![](assets/image(941).png)




 


As long as the status is not tied to any of the elements listed above, a window will be prompted seen below:   

 


![](assets/image(942).png)




 


Select Yes. The status is now deleted.
