**Category Processing**


 


Category Processing is used to perform tasks on a category of items based on rules defined by a [Configuration Object](javascript:popuplink('hs1903.htm')).  


 


To use the Category Processor, a user selects attributes which the category processor will act on and then select a target for the result of the processing.  


 


A standard configuration object for category processing feature Description Template used to create  descriptions.  The **Actions** pull-down menu lists the available configuration objects for your site.


 


The **Configure** radio button selects the mode.  When in **Configure** mode, the attributes can be selected and a processing templated created.  The **Run** mode executes the configuration object.
