**Removing a Batch of Items**


 


Removing a batch is a special function that removes a batch and all the associated items.  It is used primarily to correct a bad load of data.  


 


**BE VERY CAREFULL!  THIS FUNCTION WILL REMOVE ALL ITEMS AND DATA FOR THE SELECTED BATCH.**


 


The following conditions must exist:


 


**Only an administrator can remove a batch.**


 


To remove a batch, all the items must have the status of new.


 


All applications must be closed.


 


**To remove a batch:**


 


- Select the **Tools** drop down menu.

 


- Select **Remove Batch**.

 


- Select the batch to be deleted.

 


- Select **OK.**
