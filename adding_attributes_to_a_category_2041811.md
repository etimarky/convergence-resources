**Extended Attribute Descriptions**


 


Each 
attribute has a description field that is created and maintained in the 
Attribute Manager as a global property.  A new supplemental description 
field can now be used to add additional information on an attribute specific to 
its use on a category in the Classification Manager.  This new extended 
description information can only be viewed for the category it is used 
on.  This information does not propagate to other instances of the 
attribute.
