**Adding 
Attributes to a Category**


 


Once a category has been added, attributes can be assigned to it. 
The attributes may already be defined in the [Master Attribute List](hs2751.md). Or, new 
attributes may need to be defined for the category or item. The procedure for 
adding new attributes is explained in the **Attribute Manager** module.


 


Attributes may be added at terminal nodes or at non-terminal, 
parent nodes, and if Inherit Attributes is toggled to **Yes**, the 
attributes will flow down to the terminal nodes. 


 


**Category Developer rights are required to add attributes to 
categories.**


 


 


**To assign an attribute to a category:**


 


·                     
From the left side of the Classification Manager main window, select 
a node in the classification tree. Note that the right column of the window 
displays a detailed description of the selected category.


 


·                     
Right click on the category to which attributes will be added or 
select the Category drop down from the menu bar at the top of the window. The 
Category Maintenance menu is displayed.


 


·                     
Select **Edit Attributes…**


 


The **Attribute Mapping** 
window is displayed. From the Attribute Mapping window, the user may add 
attributes to or delete attributes from a category.


 


·                     
The Master Attribute List is displayed in alphabetical order in 
the **Master Attributes** column. Use the **scroll bar** on the right to 
move up and down through the list of attributes to find the attribute to be 
assigned.


 


·                     
Select the attribute to be assigned and press the **Assign To 
Category** button. Note that the attribute has been added to the **Attributes 
Assigned to Category** column. This column lists any attributes currently 
assigned to the selected category.


 


The [Attribute Filter](hs2400.md) may be 
used to display a shorter, more targeted list of attributes.


 


·                     
When the attribute assignment is complete, select the **Update** 
button at the lower left corner of the window.  The Attribute Mapping 
window will close, displaying the Classification Manager window. Note that the 
new attribute has been added to the attributes list in the right column.  
  



·                     
If a user, who as **Category Admin** rights, wants to add a 
attribute to other categories that are currently approved, there is another 
update option called: **Update Regardless of Status**, this will add new 
attribute to all categories that have inheritance turned on, regardless if the 
categories are approved or not.  Note: there is a similar feature for 
propagating changes to attributes through the inheritance that will also update 
approved categories.  A user has to be a Category Admin to do that as well.


 


·                     
The **Close** button will close the Attribute Mapping window, 
returning the user to the Classification Manager window. 


 


**Pressing the Close button without 
pressing the Update button will close the window without recording changes.**
